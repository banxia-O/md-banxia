## 1. 鼻咽癌 Nasopharyngeal Carcinoma (NPC)

### 治疗决策树

```
鼻咽癌（病理确诊）
├── T1N0（早期）→ 单纯根治性放疗（IMRT）
├── T1N1-3 / T2-3任何N → 同步放化疗（DDP）± 诱导化疗
├── T4任何N / 任何TN2-3（局晚）→ 诱导化疗（GP/TPF）→ 同步放化疗
└── M1（远处转移）→ 姑息化疗 ± 局部放疗
```

⚠️ 诱导化疗适应证：局部晚期（III-IVA期），尤其N2-3或T4
🔑 鼻咽癌以放疗为核心，**不做手术**（解剖位置深、放疗敏感）

### 放疗分类：根治性放疗

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-P | 原发肿瘤（鼻咽+受侵结构） | **69.96 Gy/33f** (2.12 Gy/f) |
| GTV-N | 转移淋巴结 | **66-69.96 Gy/33f** |
| CTV1（高危） | GTV-P外扩5-10mm + 高危受侵区（颅底、鼻腔、口咽、咽旁间隙）+ 咽后LN + 同侧颈II-III区 | **60 Gy/33f** (1.82 Gy/f) |
| CTV2（低危） | 双侧颈部IV-V区 + 锁骨上区（N0可不包含V区下段） | **54 Gy/33f** (1.64 Gy/f) |
| PTV | 各CTV外扩3-5mm | 同对应CTV |

🔑 鼻咽癌是**SIB（同步加量）**的经典模型
🔑 咽后淋巴结是鼻咽癌**首站转移**，必须包含在CTV1

## 2. 扁桃体癌 Tonsillar Carcinoma

### 治疗决策树

```
扁桃体癌（口咽鳞癌）
├── 早期（T1-2N0-1）
│   ├── 方案1：根治性放疗（±同步化疗）🔑 优选
│   └── 方案2：经口手术(TORS) → 根据病理决定是否术后放疗
├── 局部晚期（T3-4 或 N2-3）
│   ├── 方案1：同步放化疗（DDP）
│   └── 方案2：手术 → 术后放疗/放化疗
└── M1 → 姑息化疗 ± 免疫
```

⚠️ HPV(+)预后好，但目前标准治疗方案**不因HPV状态改变**
🔑 扁桃体癌是口咽癌最常见亚型，**对侧颈部转移率高**→双侧颈部照射

### 放疗：根治性

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-P | 原发肿瘤 | **69.96-70 Gy/33f** |
| GTV-N | 转移淋巴结 | **69.96-70 Gy/33f** |
| CTV1（高危） | GTV外扩5-10mm + 同侧口咽 + 咽后LN + 高危颈部LN区（II-III区） | **59.4 Gy/33f** |
| CTV2（低危） | 双侧颈部II-V区 + 对侧咽后LN | **54 Gy/33f** |

### 放疗：术后

| 条件 | 剂量 |
|------|------|
| 高危（切缘+和/或ECE+）| **60-66 Gy/30-33f**（瘤床），同步DDP |
| 中危（pT3-4/N2+/LVI+/PNI+）| **54-60 Gy/30f** |
| 低危区域 | **50-54 Gy** |

🔑 术后放疗启动同步化疗的指征 = **切缘阳性(positive margin)** 或 **淋巴结包膜外侵犯(ECE)**

## 3. 下咽癌 Hypopharyngeal Carcinoma

### 治疗决策树

```
下咽癌
├── 早期（T1-2N0）
│   ├── 方案1：根治性放疗（保喉）🔑 优先保喉
│   └── 方案2：手术（部分喉咽切除）
├── 局部晚期（T3-4a 或 N+）
│   ├── 喉功能保留意愿 → 诱导化疗（TPF）
│   │   ├── CR/PR → 同步放化疗（根治）
│   │   └── SD/PD → 手术（全喉+全下咽切除）→ 术后放疗
│   └── 无保喉意愿/T4b → 手术 → 术后放化疗
└── M1 → 姑息化疗 ± 免疫
```

⚠️ 下咽癌恶性度高，就诊多为晚期，预后较差
🔑 **保喉策略**：诱导化疗筛选 → 有效者放化疗，无效者手术

### 放疗：根治性（保喉）

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-P | 原发肿瘤（梨状窝/环后区/咽后壁） | **69.96-70 Gy/33-35f** |
| GTV-N | 转移淋巴结 | **69.96-70 Gy/33-35f** |
| CTV1（高危） | GTV外扩5-10mm + 同侧喉 + 下咽黏膜 + 颈部II-IV区 | **59.4 Gy/33f** |
| CTV2（低危） | 双侧颈部II-V区 + 气管食管沟LN | **54 Gy/33f** |

### 放疗：术后

| 条件 | 剂量 |
|------|------|
| 高危（切缘+/ECE+） | **60-66 Gy/30-33f**，同步DDP |
| 标准 | **54-60 Gy/30f** |

🔑 下咽癌的淋巴引流：梨状窝癌→同侧II-IV区；环后区癌→双侧II-VI区+咽后LN
🔑 与鼻咽癌/扁桃体癌剂量一致（**头颈鳞癌70 Gy统一记忆**）

## 4. NK/T细胞淋巴瘤（鼻型）Extranodal NK/T-cell Lymphoma, Nasal Type

### 治疗决策树

```
NK/T细胞淋巴瘤（鼻型）
├── 早期（I-II期，局限于上气消化道）
│   ├── 🔑 放疗为核心 + 含左旋门冬酰胺酶(L-Asp)方案化疗
│   ├── 方案1：夹心疗法（化疗-放疗-化疗）
│   ├── 方案2：序贯（化疗→放疗 或 放疗→化疗）
│   └── 方案3：同步放化疗
├── 晚期（III-IV期）
│   └── 系统化疗为主（SMILE/P-Gemox/DDGP）± 巩固放疗
└── 复发/难治 → 免疫治疗（PD-1）/异基因移植
```

⚠️ NK/T淋巴瘤对传统CHOP方案**不敏感**（P-gp介导多药耐药）
🔑 化疗核心药物 = **左旋门冬酰胺酶（L-Asp）**
🔑 **放疗对NK/T淋巴瘤极为敏感**，早期以放疗为核心

### 放疗：根治性（早期）

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV | 化疗前原发灶范围（参考治疗前MRI/PET） | **50-56 Gy/25-28f** |
| CTV | GTV + 受侵区域邻近结构（鼻腔→对侧鼻腔、筛窦、上颌窦内壁、硬腭、鼻咽） | **50 Gy/25f** |
| PTV | CTV外扩3-5mm | 同CTV |

⚠️ **累及野(involved-field)照射**，不常规预防颈部
🔑 与头颈鳞癌的区别：剂量低得多（50-56 vs 70 Gy），照射范围也更局限
🔑 化疗前范围勾画GTV（不是化疗后缩小的范围）

## 5. 脑胶质瘤 Glioma

### 治疗决策树

```
脑胶质瘤
├── 低级别胶质瘤（WHO II级/IDH突变型）
│   ├── 肿瘤<全切 或 年龄≥40 或 有高危因素
│   │   └── 术后放疗 + 化疗（TMZ或PCV）
│   └── 全切 + 无高危因素 → 密切随访
├── 高级别胶质瘤（WHO III级）
│   └── 术后 → 同步放化疗（TMZ）→ 辅助TMZ
├── 胶质母细胞瘤 GBM（WHO IV级）
│   ├── ≤70岁 KPS≥60 → Stupp方案：术后同步放化疗(60Gy+TMZ) → 辅助TMZ×6周期
│   ├── 老年/体弱 → 短程放疗(40Gy/15f) ± TMZ
│   └── KPS极差 → 最佳支持治疗
└── ⚠️ 检测MGMT启动子甲基化状态（影响TMZ疗效）
```

🔑 **Stupp方案** = GBM的金标准：同步放化疗+辅助TMZ

### 放疗：术后（GBM / 高级别胶质瘤）

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV | 术腔 + MRI T1增强残留病灶 | — |
| CTV | GTV + FLAIR/T2异常信号区 + 外扩**2 cm** | — |
| PTV | CTV外扩**3-5 mm** | **60 Gy/30f** (2 Gy/f) |

⚠️ CTV外扩需沿白质纤维束方向修剪，受脑室、颅骨、大脑镰等解剖屏障限制
🔑 GBM的CTV外扩 = **2 cm**（最大的CTV外扩距离之一，因为沿白质浸润）

### 放疗：低级别胶质瘤（WHO II级）

| 处方 | 剂量 |
|------|------|
| 标准 | **50.4 Gy/28f** 或 **54 Gy/30f** |

🔑 低级别 = 50.4-54 Gy；高级别/GBM = 60 Gy（差10 Gy，好记）

## 6. 食管癌 Esophageal Carcinoma

### 治疗决策树

```
食管癌
├── 可切除（cT1b-T3 或 cN+，可手术）
│   ├── 🔑 新辅助放化疗 → 手术（CROSS方案）—— 鳞癌+腺癌均适用
│   ├── 或 新辅助化疗 → 手术
│   └── 术后病理高危 → 可考虑术后放疗
├── 颈段食管癌
│   └── 🔑 根治性同步放化疗（不手术，保喉）
├── 不可切除/拒绝手术
│   └── 根治性同步放化疗（DDP+5-FU / 紫杉醇+DDP）
├── T1a → 内镜下切除(EMR/ESD)
└── M1 → 姑息化疗/免疫 ± 姑息放疗
```

🔑 CROSS方案 = 新辅助放化疗的里程碑试验：**41.4 Gy/23f** + 紫杉醇/卡铂

### 放疗：根治性

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-T | 食管原发肿瘤（CT/EUS/PET确定，参考钡餐+胃镜夹标记） | **60 Gy/30f** (2 Gy/f) |
| GTV-N | 转移淋巴结 | **60 Gy/30f** |
| CTV | GTV-T上下扩**3-5 cm** + 径向扩**0.5-1 cm** + 区域淋巴引流区 | **50 Gy/30f** |
| PTV | CTV外扩5-10mm | 同CTV |

⚠️ 上下扩展距离大（3-5cm），因食管黏膜下层淋巴浸润广泛

### 放疗：新辅助（CROSS方案）

| 靶区 | 处方剂量 |
|------|---------|
| GTV + 上下3cm + 径向1cm | **41.4 Gy/23f** (1.8 Gy/f) |

### 放疗：术后

| 适应证 | 剂量 |
|--------|------|
| R1/R2切缘 或 纵隔淋巴结阳性 | **50-54 Gy/25-28f** |

🔑 **数字记忆**：根治60，CROSS 41.4，术后50
🔑 食管癌CTV纵向外扩最大（3-5cm），远超其他癌种

## 7. 肺癌 Lung Cancer

### 治疗决策树

```
非小细胞肺癌 NSCLC
├── 早期可手术（I-II期）→ 手术（肺叶切除+纵隔淋巴结清扫）
│   └── 术后高危(N2/切缘+) → 辅助化疗 ± 术后放疗（PORT争议大）
├── 早期不可手术（I期）→ 🔑 SBRT
├── 局部晚期（III期）
│   ├── 可切除(IIIA选择性) → 新辅助治疗 → 手术
│   └── 不可切除(IIIB/C) → 🔑 同步放化疗 → 免疫巩固(Durvalumab)
└── 晚期（IV期）→ 系统治疗（靶向/免疫/化疗）± 姑息放疗

小细胞肺癌 SCLC
├── 局限期(LS) → 同步放化疗 → 🔑 PCI（全脑预防照射）
└── 广泛期(ES) → 化疗+免疫 ± 胸部巩固放疗 ± PCI
```

🔑 PACIFIC方案 = III期NSCLC同步放化疗后Durvalumab巩固

### 放疗：NSCLC根治性（III期同步放化疗）

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-P | 原发肺内病灶 | **60-66 Gy/30-33f** |
| GTV-N | 阳性纵隔/肺门淋巴结 | **60-66 Gy/30-33f** |
| CTV | GTV-P外扩**6-8mm** + GTV-N外扩**5mm** | 含在同一剂量水平或略低 |
| PTV | CTV外扩**5mm**（IGRT下可缩小） | 同CTV |

⚠️ 不常规预防性淋巴结照射（ENI已被累及野IFI替代）

### SBRT（早期不可手术）

| 类型 | 处方 |
|------|------|
| 周围型 | **48-54 Gy/3f** 或 **50 Gy/5f** (BED₁₀≥100 Gy) |
| 中央型 | **50 Gy/5f** 或 **60 Gy/8f**（避免过高单次剂量损伤中央结构） |

🔑 SBRT关键指标：**BED₁₀ ≥ 100 Gy** 才能保证局控

### SCLC 局限期放疗

| 方案 | 剂量 |
|------|------|
| 标准（早期同步） | **45 Gy/30f, BID**（每天2次，1.5Gy/f） |
| 替代（QD） | **60-70 Gy/30-35f** |
| PCI | **25 Gy/10f** |

🔑 SCLC放疗应**尽早**启动（化疗第1-2周期同步开始）
🔑 45/30 BID = SCLC特有的超分割方案

## 8. 乳腺癌 Breast Cancer

### 治疗决策树

```
乳腺癌
├── 早期（I-II期）
│   ├── 保乳手术(BCS) → 🔑 全乳放疗 ± 瘤床加量
│   └── 改良根治术(MRM)
│       ├── 高危(T3-4/N2-3/切缘近) → 术后胸壁+区域LN放疗
│       └── 中危(N1) → 术后胸壁+区域LN放疗（推荐）
├── 局部晚期（III期）
│   └── 新辅助化疗 → 手术 → 术后放疗
├── 炎性乳癌 → 新辅助化疗 → MRM → 术后放化疗（必须放疗）
└── IV期 → 系统治疗 ± 姑息放疗
```

⚠️ 保乳术后放疗是**标准配套**，不可省略
🔑 改良根治术后放疗指征：T3-4、N+（≥1个阳性淋巴结，尤其≥4个）、切缘阳性/近切缘

### 放疗：保乳术后

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| 全乳 | 保乳术后同侧乳房 | **50 Gy/25f** |
| 瘤床加量 | 术腔（手术夹标记）+ 外扩1-2cm | **+10-16 Gy/5-8f** |
| 大分割替代 | 全乳 | **42.56 Gy/16f** 或 **40 Gy/15f** |

🔑 大分割已成主流（≥50岁、pN0、无化疗指征的患者优选）

### 放疗：改良根治术后

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| 胸壁 | 同侧胸壁（含手术瘢痕） | **50 Gy/25f** |
| 锁骨上区 | 同侧锁骨上+下淋巴结区 | **50 Gy/25f** |
| 腋窝（选择性） | 腋窝未充分清扫时 | **50 Gy/25f** |
| 内乳区（选择性） | 内侧/中央象限 且 腋窝淋巴结阳性 | **50 Gy/25f** |

🔑 乳腺癌放疗剂量相对"低温和"（50 Gy为基础），因为是术后辅助
🔑 保乳 vs 根治术后的区别：全乳照射 vs 胸壁照射；保乳需瘤床加量

## 9. 胰腺癌 Pancreatic Cancer

### 治疗决策树

```
胰腺癌
├── 可切除 → 手术（Whipple/体尾切除）→ 辅助化疗（mFOLFIRINOX/GemCap）
│   └── 术后高危(R1/N+) → 可加术后放化疗
├── 交界可切除(BRPC) → 🔑 新辅助治疗（化疗±放疗）→ 再评估手术
├── 局部进展不可切除(LAPC)
│   └── 化疗（FOLFIRINOX/GnP）→ 评估
│       ├── 转化成功 → 手术
│       └── 稳定/进展 → 同步放化疗 或 SBRT
└── 转移性 → 姑息化疗 ± 免疫
```

⚠️ 胰腺癌手术率低（仅15-20%可切除），化疗是基础
🔑 胰腺癌放疗地位有争议，常作为化疗后的巩固/转化手段

### 放疗：术后辅助

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| CTV | 术前肿瘤范围(瘤床) + 手术吻合口 + 区域淋巴引流区 | **45-50.4 Gy/25-28f** |
| PTV | CTV外扩5-10mm（呼吸运动大） | 同CTV |

### 放疗：SBRT

| 靶区 | 处方 |
|------|------|
| GTV（可见残留/原发灶） | **33-40 Gy/5f** 或 **25 Gy/1f** |

⚠️ SBRT需注意十二指肠/胃壁剂量限制（消化道耐受差）
🔑 胰腺癌放疗剂量偏保守（常规分割45-50.4 Gy），受限于周围OAR

## 10. 宫颈癌 Cervical Cancer

### 治疗决策树

```
宫颈癌
├── 早期
│   ├── IA1（无LVSI）→ 锥切/单纯子宫切除
│   ├── IA1(LVSI+)-IA2 → 改良根治性子宫切除 + 盆腔淋巴结清扫
│   ├── IB1-IIA1（≤4cm）→ 根治性子宫切除 或 根治性放疗
│   │   └── 术后高危 → 术后放化疗
│   └── IB2-IIA2（>4cm）→ 🔑 同步放化疗（EBRT+近距离）为主
├── 局部晚期（IIB-IVA）
│   └── 🔑 根治性同步放化疗（DDP周疗）+ 近距离放疗(BT)
├── IVB（远处转移）→ 姑息化疗/免疫 ± 局部放疗
└── ⚠️ 术后Sedlis标准/Peters标准判断辅助治疗
```

🔑 Sedlis标准（中危）：LVSI/肿瘤大小/浸润深度组合 → 术后放疗
🔑 Peters标准（高危）：切缘+/淋巴结+/宫旁浸润+ → 术后同步放化疗

### 放疗：根治性（EBRT + 近距离）

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-T | 宫颈原发灶（MRI T2 + PET） | EBRT部分含在大野中 |
| GTV-N | 阳性淋巴结 | **60-65 Gy/SIB** |
| CTV | 宫颈 + 子宫体(部分) + 宫旁组织 + 阴道上段(至少3cm) + 盆腔LN区（髂总、髂外、髂内、骶前、闭孔） | **45-50.4 Gy/25-28f** |
| PTV | CTV外扩7-10mm | 同CTV |
| **近距离放疗(BT)** | 宫颈原发灶加量 | **HDR: A点 6-7 Gy × 4-5次** |
| | | 🔑 **原发灶总EQD2目标：80-90 Gy** |

⚠️ 近距离放疗OAR用 **D2cc（EQD2）** 评估（GEC-ESTRO标准）
🔑 EBRT + BT合计原发灶 **EQD2 ≥ 85 Gy** 是疗效保障的关键

### 放疗：术后

| 指征 | 靶区 | 剂量 |
|------|------|------|
| 中危(Sedlis) | 盆腔（阴道残端+宫旁+盆腔LN区） | **45-50.4 Gy/25-28f** |
| 高危(Peters) | 同上 ± 阴道残端加量 | **45-50.4 Gy** + 同步DDP |
| 腹主动脉旁LN+ | 延伸野至肾血管水平 | **45 Gy** |

## 11. 子宫内膜癌术后 Endometrial Carcinoma (Post-op)

### 治疗决策树

```
子宫内膜癌（术后分期为主）
├── 低危（IA G1-2, LVSI-）→ 观察随访
├── 中低危（IA G3 或 IB G1-2, LVSI-）→ 🔑 单纯阴道残端近距离放疗(VBT)
├── 中高危（IB G3 或 LVSI+ 或 ≥60岁合并≥2个中危因素）
│   └── VBT 或 盆腔EBRT（个体化选择）
├── 高危（II期/III期 或 非子宫内膜样/浆液性/透明细胞）
│   └── 🔑 盆腔EBRT ± VBT ± 化疗
└── IVB → 化疗为主 ± 姑息放疗
```

⚠️ 子宫内膜癌分子分型（POLE突变/MSI-H/p53突变/NSMP）正在影响治疗决策
🔑 中危的核心选择题：**VBT vs EBRT**（PORTEC-2: VBT不劣于EBRT对于中高危）

### 放疗：术后EBRT

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| CTV | 阴道残端 + 宫旁区域 + 盆腔LN区（髂总、髂外、髂内、骶前、闭孔） | **45-50.4 Gy/25-28f** |
| PTV | CTV外扩7mm | 同CTV |

### 放疗：阴道残端近距离放疗(VBT)

| 方案 | 剂量 |
|------|------|
| 单纯VBT | **HDR: 阴道黏膜面下5mm处 6 Gy × 3次** 或 **7 Gy × 3次** |
| EBRT后VBT加量 | **HDR: 4-6 Gy × 2-3次** |

🔑 子宫内膜癌 vs 宫颈癌放疗区别：
- 子宫内膜癌 = **术后**放疗为主，腔内放疗用于**阴道残端**
- 宫颈癌 = **根治性**放疗为主，腔内放疗用于**宫颈原发灶加量**

## 12. 直肠癌 Rectal Cancer

### 治疗决策树

```
直肠癌
├── 早期（T1-2N0, 距肛≥5cm）→ TME手术
├── 早期（T1N0, 局限于黏膜下层）→ 经肛局切(TEM)
├── 局部进展（T3-4 或 N+）
│   ├── 🔑 方案1（经典）：新辅助长程放化疗 → TME → 辅助化疗
│   ├── 方案2：新辅助短程放疗 → TME
│   └── 🔑 方案3（TNT全程新辅助）：
│       ├── 诱导化疗 → 短程/长程放疗 → TME
│       └── 或 短程/长程放疗 → 巩固化疗 → TME
├── 低位直肠（拟保肛）→ 新辅助治疗后cCR → Watch-and-Wait(W&W) ⚠️
└── IV期 → 系统化疗 ± 姑息放疗
```

🔑 TNT（全程新辅助治疗）是当前趋势，提高pCR率
🔑 Watch-and-Wait适用于新辅助治疗后**临床完全缓解(cCR)**的患者

### 放疗：新辅助长程放化疗

| 靶区 | 定义 | 处方剂量 |
|------|------|---------|
| GTV-T | 直肠原发肿瘤 | 含在CTV内统一剂量 |
| GTV-N | 阳性系膜/盆腔淋巴结 | 含在CTV内 |
| CTV | 直肠系膜（肿瘤上下各2-5cm）+ 骶前区 + 髂内LN区 + 闭孔LN区 ± 髂外LN(T4侵犯前方) | **50-50.4 Gy/25-28f** |
| PTV | CTV外扩5-7mm | 同CTV |
| GTV加量（选择性） | cT4/切缘受威胁的原发灶 | **+4-5.4 Gy** |

### 放疗：新辅助短程

| 方案 | 剂量 |
|------|------|
| 短程放疗 | **25 Gy/5f** (5 Gy/f) → 1周内完成 |

⚠️ 短程放疗后可等待6-8周再手术（延长间隔可提高降期率）

### 放疗：术后（未行新辅助放疗时）

| 条件 | 剂量 |
|------|------|
| 标准 | **45-50.4 Gy/25-28f** |
| 瘤床加量（R1/CRM+） | **+5.4 Gy/3f** |

🔑 直肠癌放疗的特点：**新辅助为主**，术后放疗仅限于术前未放疗的高危患者
🔑 长程50 Gy vs 短程25 Gy是最常考的对比
