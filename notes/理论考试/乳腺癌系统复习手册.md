# 乳腺癌系统复习手册

> 住培结业考试 · 放射肿瘤科 | 最后更新：2026-05-07
>
> 本文档覆盖：诊断 → 分期 → 临床表现 → 治疗决策 → 手术 → 淋巴结处理 → 化疗 → 内分泌治疗 → 靶向治疗 → 放疗（适应症/剂量/射野/OAR）→ 特殊类型 → 数字速记

---

## 一、流行病学与危险因素

- 女性最常见恶性肿瘤，发病率第一位
- 高危因素：早初潮（<12岁）、晚绝经（>55岁）、未育/晚育（>30岁）、未哺乳、肥胖、HRT（激素替代治疗）、一级亲属乳腺癌家族史、BRCA1/2突变、对侧乳癌病史、既往胸壁放疗史（如HL）
- BRCA1突变：乳腺癌+卵巢癌高风险，多为三阴性（Basal-like）
- BRCA2突变：乳腺癌（含男性乳腺癌）+卵巢癌风险

---

## 二、临床表现

### 常见表现
- **无痛性肿块**：最常见首发症状，多位于外上象限（约50%）
- 质硬、边界不清、活动度差
- 与月经周期无关（区别于乳腺增生）

### 局部晚期表现
- **皮肤改变**：酒窝征（Cooper韧带受侵）、橘皮样变（皮下淋巴管阻塞）、皮肤卫星结节、皮肤溃疡
- **乳头改变**：乳头内陷/偏斜（大导管受侵牵拉）、乳头溢液（血性多见）
- **腋窝淋巴结肿大**

### 特殊临床类型（⚠️ 高频考点）
- **炎性乳癌（IBC）**：皮肤弥漫性红肿热痛、橘皮样变，无明确肿块；病理见**真皮淋巴管癌栓**；直接分期T4d，至少ⅢB期；**禁忌先手术、禁忌保乳**
- **Paget病**：乳头湿疹样变，表皮内可见Paget细胞；>50%合并潜在导管内癌或浸润癌；分期归为Tis（Paget）

---

## 三、诊断

### 三联诊断（Triple Assessment）
1. **临床查体**
2. **影像学**：乳腺超声（年轻/致密乳腺首选）、乳腺钼靶X线（筛查金标准，40岁以上）、MRI（高危筛查、术前评估范围）
3. **病理学**：空芯针穿刺活检（CNB，首选，可判断浸润/分子分型）；细针穿刺（FNA，仅细胞学，不够）；切除活检（CNB不能明确时）

### 钼靶BI-RADS分类（了解）
- BI-RADS 0：需补充检查
- BI-RADS 1-2：阴性/良性
- BI-RADS 3：可能良性，短期随访
- BI-RADS 4：可疑，建议活检
- BI-RADS 5：高度可疑恶性
- BI-RADS 6：已确诊恶性

---

## 四、病理分型

### 非浸润性（原位癌）
- **导管原位癌（DCIS）**：局限于导管内，未穿破基底膜；分高/中/低级别；可有粉刺样坏死（高级别特征）
- **小叶原位癌（LCIS）**：双侧乳腺癌风险标志物，通常不需手术，密切随访

### 浸润性
- **浸润性导管癌（IDC）**：最常见，约70~80%
- **浸润性小叶癌（ILC）**：约10~15%，弥漫浸润生长，钼靶容易漏诊（MRI更敏感）
- 其他少见类型：髓样癌、黏液癌、管状癌（预后较好）

## 五、TNM分期（AJCC第8版）

### T分期

| T分期 | 定义 |
|-------|------|
| Tis | 原位癌（DCIS、Paget病） |
| T1 | 肿瘤最大径 ≤2cm |
| T1mi | ≤0.1cm（微浸润） |
| T1a | >0.1cm 且 ≤0.5cm |
| T1b | >0.5cm 且 ≤1cm |
| T1c | >1cm 且 ≤2cm |
| T2 | >2cm 且 ≤5cm |
| T3 | >5cm |
| T4a | 侵犯胸壁（不含胸肌） |
| T4b | 皮肤水肿/溃疡/卫星结节 |
| T4c | T4a + T4b |
| T4d | 炎性乳癌 |

> ⚠️ **T分期关键分界：2cm / 5cm**
> ⚠️ **T4 = 侵犯胸壁或皮肤，直接至少ⅢB期**

### N分期

| N分期 | 定义 |
|-------|------|
| N0 | 无区域淋巴结转移 |
| N1 | 同侧腋窝Ⅰ-Ⅱ级可活动淋巴结转移（1~3枚） |
| N2a | 同侧腋窝Ⅰ-Ⅱ级固定/融合淋巴结转移（4~9枚） |
| N2b | 临床发现同侧内乳淋巴结转移（无腋窝转移） |
| N3a | 同侧锁骨下淋巴结转移（≥10枚腋窝）或锁骨下 |
| N3b | 同侧内乳+腋窝淋巴结转移 |
| N3c | 同侧锁骨上淋巴结转移 |

> ⚠️ **N分期关键分界：3个 / 9个**
> ⚠️ **锁骨上转移 = N3c，仍属ⅢC期（非Ⅳ期！与肺癌不同）**

### M分期
- M0：无远处转移
- M1：远处转移（骨、肺、肝、脑最常见）

### 分期分组速记

| 分期 | 要点 |
|------|------|
| Ⅰ期 | T1N0M0 |
| ⅡA | T0-1N1 或 T2N0 |
| ⅡB | T2N1 或 T3N0 |
| ⅢA | T0-2N2 或 T3N1-2 |
| ⅢB | T4N0-2 |
| ⅢC | 任何T N3 |
| Ⅳ | M1 |

## 六、分子分型（⚠️ 极高频考点）

| 分子分型 | ER/PR | HER2 | Ki67 | 治疗核心 |
|----------|-------|------|------|---------|
| Luminal A | + | - | 低（<20%） | 内分泌治疗为主 |
| Luminal B（HER2-） | + | - | 高（≥20%） | 内分泌+化疗 |
| Luminal B（HER2+） | + | + | 任何 | 内分泌+化疗+抗HER2 |
| HER2过表达 | - | - | 任何 | 化疗+抗HER2 |
| 三阴性（TNBC） | - | - | 高 | 化疗（+免疫治疗） |

> ⚠️ **Ki67分界值：20%（区分Luminal A vs Luminal B）**
> ⚠️ **HER2过表达占乳腺癌约20~25%**
> ⚠️ **BRCA1突变多为三阴性**

### HER2判定标准
- IHC 3+：HER2阳性
- IHC 2+：需FISH验证（FISH扩增 = 阳性）
- IHC 0/1+：HER2阴性

## 七、治疗决策总框架

### 早期乳腺癌（Ⅰ-Ⅱ期）
**手术 → 辅助治疗（化疗/靶向/内分泌/放疗）**

### 局部晚期乳腺癌（Ⅲ期）
**新辅助化疗 → 手术 → 辅助治疗**
- 目的：降期保乳、评估药物敏感性（pCR有预后意义）

### 炎性乳癌
**新辅助化疗 → 改良根治术 → 放疗**（不保乳！）

### Ⅳ期转移性乳腺癌
**以全身治疗为主**（化疗/内分泌/靶向），局部治疗为姑息目的

---

## 八、手术

### 术式选择

| 术式 | 适应症 | 要点 |
|------|--------|------|
| **保乳手术（BCS）+ 前哨淋巴结活检（SLNB）** | 早期乳腺癌、肿瘤较小、可获得阴性切缘 | 术后必须行全乳放疗 |
| **改良根治术（MRM）** | 不适合保乳者（肿瘤大、多中心、放疗禁忌等） | 切除全乳+腋窝淋巴结清扫（ALND），保留胸肌 |

> ⚠️ **Halsted根治术（切除胸大肌+胸小肌）已基本废弃，了解即可**

### 保乳手术的绝对禁忌（⚠️ 高频考点）
- 妊娠期需放疗时（孕早期，无法推迟放疗）
- 病变广泛弥漫（弥漫恶性钙化）
- 多中心病灶（不同象限）
- 切缘反复阳性
- 既往同侧乳腺放疗史
- 炎性乳癌

> ⚠️ **不是绝对禁忌的**（考试爱出"不属于禁忌"的选项）：
> - 肿瘤位于中央区 → 不是禁忌（可以切除乳头乳晕复合体）
> - 腋窝淋巴结阳性 → 不是保乳禁忌
> - 年轻（<35岁）→ 相对禁忌，不是绝对禁忌

## 九、腋窝淋巴结处理（⚠️ 高频考点）

### 前哨淋巴结活检（SLNB）
- **适应症**：临床腋窝阴性（cN0）的患者
- **示踪方法**：蓝染料（亚甲蓝/专利蓝）和/或放射性核素（⁹⁹ᵐTc）
- **目的**：替代ALND，减少上肢淋巴水肿等并发症

### SLNB → 是否需要ALND？

| SLN结果 | 处理 | 依据 |
|---------|------|------|
| SLN阴性 | 不需要ALND | — |
| SLN 1~2枚宏转移 + **保乳** + **全乳放疗** | **可免ALND** | **Z0011研究** |
| SLN 1~2枚宏转移 + **根治术** | 需要ALND（或腋窝放疗） | Z0011不适用于根治术 |
| SLN ≥3枚阳性 | 需要ALND | — |
| SLN微转移（≤2mm） | 可免ALND | IBCSG 23-01 |
| SLN仅ITC（≤0.2mm） | 不需要ALND | — |

> ⚠️ **Z0011三个条件缺一不可：≤2枚宏转移 + 保乳手术 + 全乳放疗**
> ⚠️ 新辅助化疗后的SLN需取≥3枚，假阴性率才可接受

### 腋窝淋巴结清扫（ALND）
- 清扫范围：Ⅰ、Ⅱ水平（Berg分级），一般不常规清扫Ⅲ水平（除非Ⅱ水平有明确转移）
- 主要并发症：上肢淋巴水肿、上肢感觉异常（肋间臂神经损伤）

## 十、全身治疗

### 1. 化疗

#### 新辅助化疗适应症
- 局部晚期乳腺癌（Ⅲ期）
- 肿瘤较大希望降期保乳
- HER2+或三阴性：新辅助化疗后pCR有明确预后获益

#### 常用方案

| 方案 | 组成 | 适用 |
|------|------|------|
| **AC** | 多柔比星(A) + 环磷酰胺(C) | 基础方案 |
| **AC-T** | AC×4 → 紫杉醇(T)×4 | **最经典，蒽环序贯紫杉** |
| **TC** | 多西他赛 + 环磷酰胺 | 低危/不耐蒽环 |
| **TCbHP** | 多西他赛 + 卡铂 + 曲妥珠 + 帕妥珠 | **HER2+新辅助首选** |
| **AC-THP** | AC×4 → 紫杉醇+曲妥珠+帕妥珠 | HER2+辅助/新辅助 |

> ⚠️ **多柔比星终身累积剂量上限：450~550 mg/m²**（超过→心肌病）
> ⚠️ **蒽环 + 曲妥珠单抗不同时用**（心脏毒性叠加），序贯给药

#### 新辅助化疗后未达pCR的后续治疗
- **HER2+未达pCR** → **T-DM1（恩美曲妥珠单抗）** 14个周期（KATHERINE研究）
- **三阴性未达pCR** → **卡培他滨** 6~8个周期（CREATE-X研究）

### 2. 内分泌治疗

#### 适应症
- ER和/或PR阳性的乳腺癌

#### 药物选择

| 人群 | 药物 | 机制 |
|------|------|------|
| **绝经前** | **他莫昔芬（TAM）** | 选择性雌激素受体调节剂（SERM） |
| **绝经前高危** | **OFS（卵巢功能抑制）+ TAM/AI** | GnRHa诱导绝经后可用AI |
| **绝经后** | **芳香化酶抑制剂（AI）**：来曲唑、阿那曲唑、依西美坦 | 阻断雄激素→雌激素转化 |

> ⚠️ **用药时间：5~10年**
> ⚠️ **TAM副作用**：子宫内膜癌风险↑、血栓风险↑、潮热
> ⚠️ **AI副作用**：骨质疏松、关节痛、血脂异常
> ⚠️ **绝经前禁用AI**（除非联合OFS先诱导绝经）

### 3. 抗HER2靶向治疗

#### HER2信号通路
- HER2属于ErbB受体家族（HER1/EGFR、HER2、HER3、HER4）
- HER2**无已知配体**，通过与其他家族成员形成**异源二聚体**激活下游
- 下游通路：PI3K/AKT/mTOR（存活增殖）、RAS/RAF/MEK/ERK（增殖分化）

#### 药物

| 药物 | 作用靶点 | 要点 |
|------|---------|------|
| **曲妥珠单抗（H，赫赛汀）** | HER2胞外**结构域Ⅳ区** | 基石药物，14~18周期 |
| **帕妥珠单抗（P）** | HER2胞外**结构域Ⅱ区** | 与H不同位点，双靶联合 |
| **T-DM1（恩美曲妥珠单抗）** | ADC药物 | 新辅助未pCR后标准 |
| **T-DXd（DS-8201）** | 新一代ADC | 晚期/复发转移 |
| **拉帕替尼** | TKI小分子 | 可透过血脑屏障，脑转移可选 |

> ⚠️ **曲妥珠单抗心脏毒性：可逆（停药可恢复）**
> ⚠️ **蒽环心脏毒性：不可逆** → 两者对比是考试最爱出的
> ⚠️ 曲妥珠单抗治疗期间需**定期监测LVEF**（每3个月）
> ⚠️ 单药有效率约15~20%，联合化疗约50%以上

## 十一、放射治疗（⚠️ 重中之重）

### 1. 放疗适应症

#### 保乳术后（⚠️ 保乳必须放疗！）
- **全乳放疗（WBRT）**：所有保乳术后患者
- 瘤床加量（Boost）：浸润性癌，尤其年轻、切缘近者

#### 乳房切除术后放疗（PMRT）适应症

| 情况 | 是否放疗 |
|------|---------|
| T3（>5cm） | **是** |
| T4 | **是** |
| 腋窝淋巴结阳性≥4枚 | **是** |
| 腋窝淋巴结阳性1~3枚 | **推荐**（有高危因素时） |
| 切缘阳性/近切缘 | **是** |
| 腋窝淋巴结全阴性 + 肿瘤≤5cm | 一般不需要 |

> ⚠️ **1~3枚淋巴结阳性的PMRT是考试争议考点**：有高危因素（年轻、脉管侵犯、组织学分级高、Ki67高等）时推荐放疗

#### 区域淋巴结放疗适应症
- 腋窝淋巴结≥4枚阳性：锁骨上/下区放疗
- 1~3枚阳性伴高危因素：考虑区域淋巴结放疗
- **内乳淋巴结放疗**：有争议，非高频考点

### 2. 放疗靶区

#### 保乳术后

| 靶区 | 范围 |
|------|------|
| **全乳CTV** | 全部乳腺腺体组织 |
| **瘤床CTV（Boost区域）** | 术腔（手术夹标记）外扩1~2cm |
| **PTV** | CTV外扩0.5~1cm |

#### 乳房切除术后

| 靶区 | 范围 |
|------|------|
| **胸壁CTV** | 患侧胸壁（前界：皮下，后界：胸壁肌肉/肋骨前缘） |
| **锁骨上/下区CTV** | 锁骨上、下淋巴结区域 |
| **腋窝CTV** | 残余腋窝淋巴结区域（ALND不充分时） |

### 3. 放疗剂量（⚠️ 高频考点）

| 照射区域 | 剂量 |
|---------|------|
| **全乳放疗** | **40~50 Gy / 15~25次** |
| **大分割全乳** | **40~42.5 Gy / 15~16次**（START-B、FAST-Forward） |
| **瘤床加量（Boost）** | **10~16 Gy / 4~8次** |
| **胸壁放疗（PMRT）** | **50 Gy / 25次** |
| **锁骨上/下区** | **50 Gy / 25次** |
| **脑转移WBRT** | **30 Gy / 10次** |
| **骨转移姑息** | **30 Gy / 10次** 或 **8 Gy / 1次** |

> ⚠️ **大分割已成趋势**：START-B证实40Gy/15次等效且毒性更低
> ⚠️ **FAST-Forward**：26Gy/5次（1周完成），超大分割，疫情期间推广

### 4. 射野设计（⚠️ 高频考点）

#### 乳腺/胸壁切线野（Tangential Fields）
- **核心原理**：两个切线对穿野，避开心肺，覆盖乳腺/胸壁
- **机架角**：约15°偏向健侧（避免照对侧乳腺）
- 内切线野 + 外切线野
- **左侧乳腺**需特别注意心脏保护（心脏在左前方）
- 现代IMRT/VMAT可更好优化剂量分布

#### 射野边界（传统切线野）

| 边界 | 位置 |
|------|------|
| **上界** | 锁骨头下缘（与锁骨上野衔接） |
| **下界** | 乳腺皱襞下1~2cm |
| **内界** | 体中线（或中线旁开） |
| **外界** | 腋中线 |

#### 锁骨上野
- 前后对穿野或单前野（倾斜15°避脊髓）
- 需与切线野**无缝衔接**（半野技术/半束角技术，避免hot spot）

#### 特殊技术
- **呼吸门控/深吸气屏气（DIBH）**：左侧乳腺癌标配，心脏远离胸壁，降低心脏受量
- **俯卧位照射**：大乳房患者，减少皮肤皱褶剂量热点

### 5. 危及器官限量（Dmax/Vxx体系）

#### 保乳/胸壁放疗

| 器官 | 限量 | 备注 |
|------|------|------|
| **同侧肺** | V20 < 25~30%，V5 < 50~55% | 放射性肺炎风险 |
| **心脏（左侧）** | V25 < 10%，Dmean < 4~8 Gy | DIBH可显著降低 |
| **心脏（右侧）** | Dmean越低越好 | 右侧心脏受量本身较低 |
| **对侧乳腺** | Dmax < 3~5 Gy | 远期继发肿瘤风险 |
| **对侧肺** | V5 < 10% | — |
| **脊髓** | Dmax < 45 Gy | 锁骨上野涉及时注意 |
| **臂丛神经** | Dmax < 60~66 Gy | 锁骨上区照射涉及 |
| **甲状腺** | 尽量减少 | 锁骨上区照射涉及 |
| **LAD（左前降支）** | Dmean尽量<10 Gy | 新兴关注器官 |

> ⚠️ **左侧乳腺癌心脏保护三板斧**：DIBH + IMRT + 心脏限量严格控制
> ⚠️ **同侧肺V20是最常考的OAR限量指标**

## 十二、特殊类型乳腺癌

### 1. 炎性乳癌（IBC）
- T4d，至少ⅢB期
- 病理：**真皮淋巴管癌栓**
- 治疗：**新辅助化疗 → 改良根治术 → 放疗**
- ⚠️ **禁忌保乳，禁忌先手术**

### 2. DCIS（导管原位癌）
- 保乳术后需放疗（降低同侧复发50%）
- 不需要化疗，不需要ALND
- ER+的DCIS可用TAM降低对侧风险
- 低危DCIS可考虑单纯切除（VNPI评分系统）

### 3. 妊娠期乳癌
- 手术：可以做，优先改良根治术（不能保乳→因为不能放疗）
- 化疗：孕中期后可用AC方案；**禁用紫杉醇、甲氨蝶呤**
- 放疗：**孕期绝对禁忌**，推迟到产后
- 内分泌：TAM、AI均禁用
- 抗HER2：曲妥珠单抗禁用（FDA孕期D类）

### 4. 男性乳腺癌
- 罕见，与BRCA2突变相关
- 多为ER+
- 治疗原则同女性，手术首选改良根治术
- 内分泌治疗首选TAM（男性不用AI）

### 5. 隐匿性乳腺癌
- 腋窝淋巴结转移为首发表现，乳房查体和影像学未发现原发灶
- MRI有助于发现隐匿原发灶
- 找不到原发灶：按乳腺癌治疗（ALND+全乳放疗或改良根治术）

## 十三、转移性乳腺癌要点

### 常见转移部位
- **骨转移**：最常见远处转移部位
- **肺转移**
- **肝转移**
- **脑转移**：HER2+和三阴性更容易脑转移

### 转移处理
- 骨转移首选：**放疗**（姑息止痛）+ 双膦酸盐/地诺单抗
- 脑转移首选：**全脑放疗（WBRT）**30Gy/10次 或 SRS（≤3个病灶、每个<3cm时考虑）
- 拉帕替尼可透过血脑屏障（HER2+脑转移可选）

## 十五、高频考试陷阱清单

1. **保乳术后必须放疗**——保乳不放疗不是保乳
2. **Z0011三个条件**：≤2枚SLN宏转移 + 保乳 + 全乳放疗 → 可免ALND；根治术患者不适用
3. **炎性乳癌不保乳、不先手术**，新辅助化疗先行
4. **蒽环心脏毒性不可逆 vs 曲妥珠单抗心脏毒性可逆**——对比题最爱出
5. **绝经前禁用AI**（除非联合OFS先诱导绝经）
6. **锁骨上淋巴结转移 = N3c = ⅢC期**，不是Ⅳ期！
7. **妊娠期不能放疗、不能用TAM/AI/曲妥珠单抗**
8. **内乳淋巴结放疗有争议**，考试一般不会深入考
9. **切线野约15°偏向健侧**，不是对称的
10. **左侧乳腺癌必须考虑心脏保护**（DIBH是标配技术）
11. **大分割放疗已有充分证据**（START-B、FAST-Forward），不是实验性的
12. **骨转移、脑转移首选放疗**
13. **HER2+新辅助未达pCR → T-DM1**（KATHERINE）；**三阴性未达pCR → 卡培他滨**（CREATE-X）
14. **男性乳腺癌用TAM不用AI**
15. **DCIS不需要化疗、不需要ALND**
