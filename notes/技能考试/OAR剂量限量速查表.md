# 危及器官（OAR）剂量限量速查表

> 适用于常规分割（1.8–2.0 Gy/次）外照射，特殊分割方案需换算。
> 格式：Dmax / Dmean / Vxx，剂量单位 Gy，体积单位 %。
> 🔑 = 高频考点

## 六、速记口诀 🔑

- **脊髓45，脑干54**：最基础的两个数，必须秒答
- **晶体7**：全身最低耐受OAR
- **腮腺26，至少保一侧**：头颈部保护唾液功能
- **肺V20 < 30%，Dmean < 20**：放射性肺炎预测
- **心脏V30 < 40%，左乳Dmean < 8**：心脏保护两套标准
- **肝Dmean < 30，肾V20 < 33%**：腹部实质脏器
- **小肠54，股骨头V50 < 5%**：盆腔高频
- **膀胱直肠V50 < 50%**：盆腔照射基本线
- **骨盆骨髓V10/V20/V40**：同步化疗时才特别关注
