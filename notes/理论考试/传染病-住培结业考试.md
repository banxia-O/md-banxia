## 一、病毒性肝炎 Viral Hepatitis

### 1.1 病原学对比（超高频考点）

| 项目 | HAV（甲肝） | HBV（乙肝） | HCV（丙肝） | HDV（丁肝） | HEV（戊肝） |
|------|-------------|-------------|-------------|-------------|-------------|
| 核酸类型 | RNA | **DNA**（唯一DNA肝炎病毒） | RNA | RNA | RNA |
| 传播途径 | 粪-口 | 血液/体液/母婴 | 血液（输血最多见） | 同HBV | 粪-口 |
| 慢性化 | **不慢性化** | 成人5%~10%，新生儿**90%** | **最易慢性化（～80%）** | 加重HBV病情 | **不慢性化**（孕妇易重型） |
| 疫苗 | 有（灭活疫苗） | 有（基因工程疫苗） | **无** | 预防HBV即可预防 | 有（中国研发） |
| 肝癌风险 | 无 | **有** | **有** | 有 | 无 |

**速记口诀**：
- "甲戊粪口不慢性"——传播途径和预后一起记
- "丙肝最慢性、最没疫苗"——两个"最"
- HBV是唯一DNA病毒——独一份
- 孕妇+戊肝=重型高发——送命组合

### 1.2 HBV血清学标志物（两对半）——必考

| 序号 | 标志物 | 缩写 | 临床意义 |
|------|--------|------|----------|
| 1 | 乙肝表面抗原 | HBsAg | HBV感染标志，最早出现 |
| 2 | 乙肝表面抗体 | 抗-HBs (HBsAb) | **保护性抗体**，疫苗接种成功/既往感染恢复 |
| 3 | e抗原 | HBeAg | 病毒复制活跃，**传染性强** |
| 4 | e抗体 | 抗-HBe (HBeAb) | 病毒复制减弱（但不代表无传染性） |
| 5 | 核心抗体 | 抗-HBc (HBcAb) | 既往或现症感染；**IgM型提示近期/急性感染**，IgG型提示既往感染 |

**经典模式辨析**：

| 模式 | 阳性项 | 含义 |
|------|--------|------|
| **大三阳** | HBsAg、HBeAg、抗-HBc（1,3,5） | 病毒复制活跃，传染性强 |
| **小三阳** | HBsAg、抗-HBe、抗-HBc（1,4,5） | 病毒复制减弱（需结合HBV-DNA判断） |
| 恢复期 | 抗-HBs、抗-HBe、抗-HBc（2,4,5） | 既往感染已恢复，有免疫力 |
| 疫苗接种后 | **仅抗-HBs阳性**（仅2） | 接种成功，有保护力 |
| 窗口期 | **仅抗-HBc阳性**（仅5） | HBsAg已消失、抗-HBs尚未出现 |

**注意陷阱**：
- 小三阳≠病毒不复制，必须查**HBV-DNA**才能判断
- HBV-DNA是反映病毒复制的**最直接、最可靠**指标
- cccDNA是HBV在肝细胞内持续存在的分子基础，是慢性化和治疗困难的根源
- **抗-HBc不能通过疫苗产生**，出现即代表有过自然感染

### 1.3 临床分型

```
病毒性肝炎
├── 急性肝炎
│   ├── 急性黄疸型
│   └── 急性无黄疸型
├── 慢性肝炎（病程>6个月）
│   ├── 轻度：ALT轻度异常，症状轻
│   ├── 中度：症状+体征+ALT明显异常
│   └── 重度：有明显肝功能失代偿表现但未达重型标准
├── 重型肝炎（肝衰竭）⭐
│   ├── 急性重型（暴发性）：≤2周
│   ├── 亚急性重型：15天~24周
│   └── 慢加急性/慢性重型：慢性肝病基础上
├── 淤胆型肝炎
└── 肝炎肝硬化
```

### 1.4 急性肝炎临床表现（三期）

**① 黄疸前期（前驱期）**
- 乏力、纳差、厌油、恶心呕吐（消化道症状突出）
- 可有低热、肝区隐痛
- 尿色加深（最早注意到的症状之一）
- 此期传染性**最强**（尤其甲肝）
- 持续5~7天

**② 黄疸期**
- 巩膜→皮肤黄染，尿色深如浓茶
- 肝大，有压痛和叩击痛
- 部分患者脾大
- ALT达峰后开始下降，TBil继续升高
- 持续2~6周

**③ 恢复期**
- 黄疸消退，症状减轻
- 肝功能逐渐恢复
- 持续1~3个月

### 1.5 重型肝炎（肝衰竭）——重点中的重点 ⭐

**诊断标准核心指标**：
- **PTA（凝血酶原活动度）< 40%** ——最重要的判断指标
- TBil ≥ 171 μmol/L（正常上限10倍）或每日升高 ≥ 17.1 μmol/L
- 极度乏力+严重消化道症状
- 出血倾向明显
- 肝性脑病

**急性重型 vs 亚急性重型**：

| 特征 | 急性重型（暴发性） | 亚急性重型 |
|------|-------------------|-----------|
| 起病至肝衰竭时间 | ≤ 2周 | 15天 ~ 24周 |
| 脑病 | 早期出现（Ⅱ度以上） | 后期出现 |
| 肝脏 | 进行性缩小 | 可不缩小 |
| 腹水 | 少见 | 明显 |
| 胆酶分离 | **有**（TBil升高但ALT下降） | 可有 |
| 预后 | 极差（病死率>70%） | 较差 |

**胆酶分离**：TBil持续升高而ALT反而下降——表面看"好转"，实则肝细胞大量坏死已无酶可释放，是**预后极差**的征兆。

### 1.6 实验室检查要点

| 检查项目 | 临床意义 |
|----------|----------|
| **ALT** | 肝细胞损伤最**敏感**指标，急性期最早升高 |
| AST | 线粒体来源为主；AST/ALT > 1 提示慢性化或重症 |
| TBil | 反映黄疸程度；直接胆红素升高为主→肝细胞性黄疸 |
| 白蛋白 | 反映肝脏合成功能，半衰期21天，急性肝炎可正常，慢性肝病时降低 |
| **PTA / INR** | 反映凝血因子合成能力，**判断重型肝炎最关键指标** |
| 胆碱酯酶 | 降低提示肝实质损害严重程度 |
| AFP | 急性肝炎时一过性升高→提示肝细胞再生（非肿瘤标志） |

**肝炎→肝癌的鉴别**：AFP持续升高+影像学占位→高度怀疑肝癌；一过性升高伴ALT恢复→肝细胞再生。

### 1.7 治疗原则

#### 急性肝炎
- 休息+保肝对症为主
- 甲肝、戊肝多可自愈，一般不需抗病毒

#### 慢性乙肝抗病毒治疗（重点）

**抗病毒适应证**：
- HBV-DNA阳性 **+** ALT持续异常或肝组织学有炎症/纤维化证据
- 肝硬化患者：只要HBV-DNA阳性即需抗病毒，不论ALT

**一线药物**：

| 药物 | 类型 | 特点 |
|------|------|------|
| 恩替卡韦 (ETV, Entecavir) | 核苷类似物（NAs） | 强效低耐药，口服，长期用药 |
| 替诺福韦 (TDF, Tenofovir) | 核苷酸类似物（NAs） | 强效低耐药，口服，注意肾毒性 |
| 丙酚替诺福韦 (TAF) | 核苷酸类似物（NAs） | TDF改良版，肾脏和骨骼安全性更好 |
| 聚乙二醇干扰素α (Peg-IFNα) | 免疫调节 | 有限疗程（48周），实现功能性治愈可能性更高；禁忌证多（失代偿肝硬化禁用） |

**NAs vs 干扰素选择要点**：
- 失代偿肝硬化→**只能用NAs**，干扰素禁用
- 妊娠→替诺福韦（TDF）相对安全（FDA妊娠B级）
- 追求停药→干扰素更有可能实现HBsAg阴转

#### 慢性丙肝
- DAAs（直接抗病毒药物）：索磷布韦/维帕他韦(Sofosbuvir/Velpatasvir)等
- 泛基因型方案，治愈率 > 95%
- **丙肝现在可以治愈**——这是考点

#### 重型肝炎
- 综合支持治疗（维持内环境稳定）
- 人工肝支持系统
- 肝移植（终极治疗手段）
- 禁用损肝药物

### 1.8 预防

| 类型 | 疫苗 | 关键点 |
|------|------|--------|
| 甲肝 | 灭活疫苗/减毒活疫苗 | 儿童常规接种 |
| **乙肝** | 基因工程疫苗 | **0-1-6方案**（出生、1月龄、6月龄） |
| 丙肝 | **无疫苗** | 血液筛查+减少暴露 |
| 丁肝 | 预防HBV即可预防HDV | — |
| 戊肝 | 有（益可宁®） | 中国研发，全球首个 |

**乙肝母婴阻断**（必考）：
- 新生儿出生后 **12小时内** 注射 HBIG（乙肝免疫球蛋白）+ 乙肝疫苗第1针
- 后续按0-1-6完成全程接种
- 高病毒载量母亲（HBV-DNA > 2×10⁵ IU/mL）：孕晚期（28周起）口服TDF降低宫内感染风险

### 1.9 与放疗科交叉考点

- 放疗/化疗前需筛查HBV：HBsAg阳性患者化疗期间可致HBV再激活→重型肝炎
- 预防性抗病毒：HBsAg阳性患者在免疫抑制治疗（化疗、靶向、免疫治疗）前应**预防性**使用ETV/TDF
- 即使HBsAg阴性但抗-HBc阳性，使用利妥昔单抗等强免疫抑制剂时也需监测HBV-DNA
- 肝脏放疗时需关注肝功能基线状态，合并肝炎者OAR限量需更保守

## 二、麻疹 Measles

### 2.1 病原学

- 麻疹病毒（Measles virus），副黏液病毒科，单股负链RNA病毒
- **只有一个血清型**——感染或接种后可获**持久免疫**（理论上终身）
- 对外界抵抗力弱：对热、紫外线、消毒剂敏感，离体后很快失活
- 传染性极强（R₀ ≈ 12~18，远超流感和新冠）

### 2.2 流行病学

- **传染源**：患者是唯一传染源
  - 传染期：发病前2天 ~ 出疹后5天（合并肺炎延长至出疹后10天）
  - 前驱期（卡他期）传染性**最强**
- **传播途径**：呼吸道飞沫传播
- **易感人群**：未接种疫苗/未感染过的人群
  - < 6月龄婴儿因母传抗体保护，较少发病
  - 8月龄后母传抗体消失→易感窗口期
- **流行特征**：冬春季多发

### 2.3 发病机制（了解即可）

```
病毒吸入 → 呼吸道上皮增殖 → 局部淋巴结
→ 第一次病毒血症（前驱期）→ 全身单核-巨噬细胞系统
→ 第二次病毒血症 → 皮肤黏膜出疹 + 全身多器官受累
```

**关键点**：麻疹可导致**暂时性细胞免疫抑制**（持续数周~数月），因此易继发感染，且可使已阳转的PPD试验**暂时转阴**（结核感染者注意！）

### 2.4 临床表现（三期）——高频考点 ⭐

#### ① 前驱期（卡他期）：3~4天

- **发热**：中~高热，逐渐升高
- **上呼吸道卡他症状**：咳嗽、流涕、喷嚏（"感冒样"表现）
- **结膜炎**：畏光、流泪、结膜充血、眼睑水肿
- **Koplik斑（麻疹黏膜斑）**⭐⭐⭐
  - 出疹前**1~2天**出现
  - 位置：双侧**第二磨牙对面的颊黏膜**
  - 形态：针尖大小**灰白色小点**，周围有红晕，初为数个后可融合
  - 意义：麻疹**早期诊断的特征性体征**（具有早期诊断价值）
  - 消失时间：出疹后1~2天迅速消失
- 此期传染性**最强**

#### ② 出疹期：3~5天

- 发热第**3~4天**开始出疹，出疹时体温再次升高达峰（40~40.5°C）
- **出疹顺序**（必考）：**耳后、发际 → 面部 → 颈部 → 躯干 → 四肢 → 手掌足底**
  - 概括：从上到下，从中心到外周
  - 约2~3天出齐
- **皮疹特点**：
  - 红色斑丘疹，大小不等，压之褪色
  - 可相互融合，**疹间皮肤正常**
  - 重症可呈出血性（暗红色瘀点，预后差）
- 全身症状加重：高热、咳嗽、精神萎靡

#### ③ 恢复期

- 出疹3~5天后，皮疹按出疹顺序依次消退
- 体温下降，症状减轻
- 消退后留有**棕褐色色素沉着 + 糠麸样脱屑**
  - 这是与风疹、幼儿急疹的重要鉴别点！

**速记**：麻疹出疹 = "发热3天出疹，出疹3天出齐，退疹3天退净"——三个3

### 2.5 非典型麻疹（考试陷阱）

| 类型 | 特点 |
|------|------|
| 轻型麻疹 | 接种过疫苗/被动免疫后，症状轻，Koplik斑不典型，出疹少且散，容易漏诊 |
| 重型麻疹 | 中毒性（高热惊厥）、出血性（皮疹出血、DIC）、疱疹性 |
| 成人麻疹 | 中毒症状重，但并发症相对少 |

### 2.6 并发症——高频考点 ⭐

**① 肺炎**——**最常见的并发症，也是麻疹最主要的死亡原因**
- 麻疹病毒性肺炎：出疹期发生
- 继发细菌性肺炎：更常见，出疹后期或恢复期
- 巨细胞肺炎（Hecht肺炎）：免疫缺陷者，预后极差

**② 喉炎**
- 表现：声嘶、犬吠样咳嗽、吸气性喉鸣、三凹征
- 2~3岁小儿多见
- 可致喉梗阻危及生命

**③ 脑炎**
- 急性麻疹脑炎：出疹后2~6天，发生率约0.1%~0.2%
- **亚急性硬化性全脑炎（SSPE）**⭐：
  - 麻疹感染后**数年**（平均5~7年）发病
  - 表现：进行性智力减退 + 行为异常 + 肌阵挛 + 去大脑强直
  - 属于慢性持续性感染（慢病毒感染）
  - 脑脊液和血清中**麻疹抗体滴度极高**
  - 预后极差，几乎100%死亡

**④ 结核恶化**
- 麻疹的免疫抑制可使潜伏结核活化、播散
- 原已阳性的PPD可暂时转阴

**⑤ 营养不良与维生素A缺乏**
- 发展中国家麻疹死亡率高与营养状态密切相关

### 2.7 诊断

**临床诊断**（多数靠临床即可）：
- 流行病学接触史（接触患者后10~14天发病）
- 前驱期卡他症状 + Koplik斑
- 典型出疹顺序 + 退疹后色素沉着脱屑

**实验室确诊**：
- **麻疹特异性IgM抗体**（出疹后3天开始阳性）——最常用确诊手段
- 恢复期与急性期IgG抗体≥4倍升高
- 咽拭子/尿液病毒分离或核酸检测

### 2.8 鉴别诊断——出疹性疾病对比表（必背）⭐⭐⭐

| 项目 | 麻疹 | 风疹 | 幼儿急疹 | 猩红热 | 水痘 |
|------|------|------|----------|--------|------|
| 病原 | 麻疹病毒 | 风疹病毒 | HHV-6 | A组乙型溶血性链球菌 | VZV |
| 好发年龄 | 6月~5岁 | 儿童/青少年 | **6月~2岁** | 2~10岁 | 儿童 |
| 发热与出疹关系 | 发热3~4天出疹 | 发热1~2天出疹 | **热退疹出** | 发热1~2天出疹 | 发热同日或次日出疹 |
| 出疹顺序 | 耳后→面→躯干→四肢 | 面→躯干→四肢（**1天出齐**） | 躯干→面部四肢 | 颈胸→全身（**1天出齐**） | 躯干→面部四肢（**向心性分布**） |
| 皮疹形态 | 红色斑丘疹，可融合 | 浅红色细小斑丘疹，不融合 | 红色斑丘疹 | 弥漫充血+密集针尖样丘疹，**皮肤皱褶处线状疹（帕氏线Pastia线）** | **斑疹→丘疹→疱疹→结痂**（四世同堂） |
| 退疹后表现 | **色素沉着 + 糠麸样脱屑** | 无色素沉着，无脱屑 | 无色素沉着 | **大片脱皮** | 一般不留瘢痕（除非继发感染） |
| 特征性表现 | **Koplik斑** | **耳后枕后淋巴结肿大** | **热退疹出** | **草莓舌/杨梅舌**、口周苍白圈 | 向心性分布，各期皮疹同时存在 |
| 主要并发症 | 肺炎 | 先天性风疹综合征（孕妇） | 惊厥 | 风湿热、肾小球肾炎 | 皮肤继发感染、脑炎 |

**鉴别记忆技巧**：
- 退疹后有色素沉着的 → **只有麻疹**
- 热退疹出的 → **只有幼儿急疹**
- 1天出齐的 → 风疹、猩红热
- 向心性分布 + 四世同堂 → 水痘
- 口周苍白圈 + 杨梅舌 → 猩红热

### 2.9 治疗

麻疹**无特效抗病毒药物**，以对症支持为主：

- **一般治疗**：卧床休息，通风良好，充足水分和营养
- **对症处理**：
  - 高热：物理降温为主，慎用大量退热药（避免虚脱+皮疹内陷）
  - 咳嗽：祛痰为主，剧烈咳嗽可用镇咳药
  - 继发细菌感染：使用抗生素
- **维生素A**⭐：WHO推荐，可减少麻疹并发症和死亡率
  - 6月~1岁：10万IU，口服×2天
  - ≥1岁：20万IU，口服×2天
- **并发症处理**：
  - 肺炎：抗感染+支持
  - 喉炎：雾化吸入、必要时气管切开
  - 脑炎：脱水降颅压、对症

### 2.10 预防

**主动免疫**：
- **麻疹减毒活疫苗**（含MMR/MR联合疫苗）
- 初种：**8月龄**（中国国家免疫规划）
- 复种：**18月龄**
- 接种禁忌：免疫缺陷、活动性结核、孕妇（减毒活疫苗通用禁忌）

**被动免疫**：
- 接触麻疹后**5天内**注射人免疫球蛋白 → 可预防发病或减轻病情
- 适用于：体弱、免疫低下、未接种疫苗的接触者

**隔离**：
- 患者隔离至出疹后**5天**（合并肺炎延长至出疹后**10天**）
- 密切接触者医学观察**21天**（最长潜伏期）

### 2.11 与放疗科交叉考点

- 化疗/放疗致免疫抑制 → 麻疹感染后病情更重，并发巨细胞肺炎风险高
- 免疫抑制患者接触麻疹后应尽早注射免疫球蛋白
- 免疫抑制状态下**禁用麻疹减毒活疫苗**
- 既往接种过麻疹疫苗的患者，化疗后可能抗体消失 → 化疗结束免疫重建后需考虑**再次接种**

## 三、病毒性脑炎 Viral Encephalitis

> 考试重点：流行性乙型脑炎（乙脑）为核心，单纯疱疹病毒脑炎（HSE）为补充；脑脊液鉴别诊断是必考表格。

### 3.1 流行性乙型脑炎 Japanese Encephalitis (JE) ⭐⭐⭐

#### 3.1.1 病原学
- 乙脑病毒（JEV），**黄病毒科**，单股正链RNA
- 不耐热，56°C 30min灭活

#### 3.1.2 流行病学——考点密集

| 要素 | 内容 |
|------|------|
| 传染源 | **猪**是最重要的传染源和扩增宿主（猪感染率高→蚊叮猪→蚊叮人） |
| 传播媒介 | **三带喙库蚊**（主要传播媒介） |
| 传播途径 | 蚊虫叮咬（虫媒传播），**人→人不直接传播** |
| 流行季节 | **夏秋季**（7~9月），与蚊虫活动高峰吻合 |
| 好发年龄 | **10岁以下儿童**多见（成人多已隐性感染获得免疫） |
| 感染类型 | 绝大多数为**隐性感染**（显性:隐性 ≈ 1:300~2000） |

**速记**：猪-蚊-人，夏秋季，小儿多，隐性感染比例极高

#### 3.1.3 临床表现（四期）

**① 初期（1~3天）**
- 急起发热、头痛、恶心呕吐
- 嗜睡、精神萎靡

**② 极期（第4~10天）**——决定预后的关键阶段

三大核心表现（必考）：

| 表现 | 特点 |
|------|------|
| **高热** | 体温常 ≥ 40°C，热程长，热度越高预后越差 |
| **意识障碍** | 嗜睡→昏睡→昏迷，意识障碍程度是判断病情轻重的主要依据 |
| **惊厥/抽搐** | 可反复发作，全身性或局限性，是病情严重的标志 |

其他表现：
- 脑膜刺激征：颈项强直、Kernig征（+）、Brudzinski征（+）
- 颅内压增高：头痛加重、喷射性呕吐、视乳头水肿
- **呼吸衰竭**：中枢性呼吸衰竭 →**最主要的死亡原因**
  - 表现：呼吸节律不齐、深浅不一、双吸气、叹息样呼吸、潮式呼吸、呼吸暂停
- 可有病理反射（巴宾斯基征阳性）

**③ 恢复期**
- 约病程第2周，体温下降，神经症状逐渐恢复
- 重症恢复慢，可达数周~数月

**④ 后遗症期**
- 约5%~20%有后遗症（重型可达30%以上）
- 常见后遗症：失语、瘫痪（肢体）、精神异常、智力减退、癫痫
- 部分可恢复（半年~2年内），部分永久残留

#### 3.1.4 临床分型

| 分型 | 体温 | 意识 | 抽搐 | 呼吸衰竭 | 后遗症 |
|------|------|------|------|----------|--------|
| 轻型 | 38~39°C | 清醒或嗜睡 | 无或偶有 | 无 | 无 |
| 普通型 | 39~40°C | 浅昏迷 | 短暂抽搐 | 无 | 少 |
| **重型** | ≥ 40°C | 昏迷 | 反复抽搐 | 可有 | 常见 |
| **极重型** | ≥ 41°C | 深昏迷 | 频繁/持续抽搐 | **有** | 多且重/死亡 |

**考试判断技巧**：题目给体温+意识状态+有无抽搐+有无呼衰 → 直接定型

#### 3.1.5 实验室检查

**血常规**：
- WBC升高（10~20×10⁹/L），**中性粒细胞为主**

**脑脊液**（见3.3详细对比表）：
- 压力增高
- 外观清亮或微混
- WBC轻~中度升高（50~500×10⁶/L）
- **早期中性粒细胞为主，2~3天后转为淋巴细胞为主**——这个转变过程是考点
- 蛋白轻度升高
- 糖**正常**、氯化物**正常**

**确诊**：
- 血清特异性**IgM抗体**（发病3~4天即可阳性）——最常用快速诊断方法
- 恢复期与急性期IgG抗体 ≥ 4倍升高

#### 3.1.6 治疗——无特效抗病毒药

核心原则：**降温、止惊、降颅压 + 防治呼吸衰竭**

| 治疗措施 | 具体方法 |
|----------|----------|
| 降温 | 物理降温为主（冰帽、冰毯、酒精擦浴），药物退热辅助 |
| 止惊 | **地西泮**（首选，静注）；苯巴比妥；水合氯醛灌肠 |
| 降颅压 | **20%甘露醇**快速静滴（半小时内），每6~8h一次 |
| 呼吸衰竭 | 保持气道通畅、吸氧、呼吸兴奋剂；必要时气管插管+机械通气 |

**呼吸衰竭是抢救的关键**——处理好呼衰就保住了命

#### 3.1.7 预防

- **灭蚊防蚊**——最根本的措施
- 管理好猪等家畜（传染源控制）
- **疫苗接种**：
  - 乙脑灭活疫苗：8月龄2针（间隔7~10天），2岁和6岁各加强1针
  - 乙脑减毒活疫苗：8月龄1针，2岁加强1针

### 3.2 单纯疱疹病毒脑炎 Herpes Simplex Encephalitis (HSE)

> 散发性脑炎中**最常见**的类型，也是**唯一有特效抗病毒药**的病毒性脑炎

| 项目 | 要点 |
|------|------|
| 病原 | **HSV-1**（成人）；新生儿可为HSV-2 |
| 发病机制 | 病毒沿嗅神经/三叉神经逆行至脑，主要侵犯**颞叶**（额叶底面也可受累） |
| 临床特点 | 急性起病，高热、头痛、意识障碍、**精神行为异常**（幻觉、人格改变）、**记忆障碍**、癫痫发作 |
| 颞叶受累线索 | 精神症状突出、嗅觉异常、颞叶癫痫样发作 |
| 脑脊液 | 压力升高，淋巴细胞为主，蛋白升高，糖正常，**可见红细胞**（出血坏死性改变） |
| 影像学 | CT/MRI示**颞叶**低密度/异常信号灶（常不对称），可有出血 |
| 确诊 | 脑脊液**HSV-DNA PCR**（金标准）；脑活检 |
| 治疗 | **阿昔洛韦 (Acyclovir)**⭐——特效药，越早用预后越好，疗程14~21天 |

**考试抓手**：颞叶受累 + 精神症状 + 脑脊液见红细胞 → 想到HSE → 阿昔洛韦

---

### 3.3 脑脊液鉴别诊断对比表——必背 ⭐⭐⭐

| 项目 | 正常值 | 化脓性脑膜炎 | 病毒性脑炎/脑膜炎 | 结核性脑膜炎 | 隐球菌脑膜炎 |
|------|--------|-------------|-------------------|-------------|-------------|
| **外观** | 清亮 | **混浊/脓性** | 清亮或微混 | **毛玻璃样**（静置后形成薄膜） | 清亮或微混 |
| **压力** | 70~180 mmH₂O | **显著增高** | 正常或轻度增高 | 增高 | 显著增高 |
| **WBC** | 0~5×10⁶/L | **显著增高（>1000）** | 轻~中度增高（<500） | 中度增高（<500） | 轻~中度增高 |
| **细胞分类** | — | **中性粒细胞为主** | **淋巴细胞为主** | **淋巴细胞为主** | 淋巴+单核为主 |
| **蛋白** | 0.15~0.45 g/L | **显著增高** | 正常或轻度增高 | **显著增高** | 增高 |
| **糖** | 2.5~4.4 mmol/L | **显著降低** | **正常** | **显著降低** | **降低** |
| **氯化物** | 120~130 mmol/L | 降低 | **正常** | **显著降低（最明显）** | 降低 |

#### 鉴别速记口诀

**看糖定大类**：
- 糖正常 → **病毒性**（病毒不吃糖）
- 糖降低 → 细菌性 or 结核性 or 真菌性（微生物消耗葡萄糖）

**糖降低后再分**：
- 外观脓性 + WBC极高（>1000）+ 中性粒为主 → **化脓性脑膜炎**
- 毛玻璃样 + 薄膜形成 + **氯化物最低** + 慢性起病 → **结核性脑膜炎**
- 墨汁染色找到隐球菌 → **隐球菌脑膜炎**

**其他鉴别线索**：

| 疾病 | 独有线索 |
|------|----------|
| 化脓性脑膜炎 | 涂片/培养找到致病菌；婴儿常见脑膜炎双球菌、肺炎链球菌、流感嗜血杆菌 |
| 结核性脑膜炎 | 慢性起病（数周）、脑神经麻痹（Ⅵ对最常受累）、脑底病变为主、PPD/T-SPOT、抗酸染色 |
| HSE | 颞叶病变、精神症状、脑脊液可见**红细胞** |
| 隐球菌脑膜炎 | **墨汁染色**找到带荚膜酵母菌、常见于免疫缺陷患者（AIDS） |

### 3.4 与放疗科交叉考点

- 脑转移瘤放疗后出现发热+神经症状 → 需鉴别**放射性脑损伤 vs 颅内感染**
  - 放射性脑坏死：多在放疗后数月~数年，MRI可见增强灶（与肿瘤复发也需鉴别），脑脊液改变轻
  - 颅内感染：脑脊液改变明显，结合病原学检查
- 化疗/放疗后免疫抑制 → 机会性感染风险增加：
  - 隐球菌脑膜炎（CD4低时）
  - CMV脑炎
  - 弓形虫脑炎
- 鞘内注药（如MTX）后出现脑膜刺激征 → 需排除化学性脑膜炎（药物性 vs 感染性）

## 四、细菌性痢疾 Bacillary Dysentery

### 4.1 病原学

- **志贺菌属（Shigella）**，G⁻杆菌，无鞭毛（不运动）、无荚膜、不形成芽胞
- 致病物质：侵袭力（侵入结肠上皮细胞）+ 内毒素 + 外毒素（志贺毒素，仅A群产生）
- 四群分类及考试要点：

| 群 | 名称 | 考试关键词 |
|----|------|-----------|
| A群 | 痢疾志贺菌 | **毒力最强**，可产志贺毒素（Shiga toxin），易致中毒型和重型 |
| B群 | 福氏志贺菌 | **中国最常见**的菌型，最易转为**慢性** |
| C群 | 鲍氏志贺菌 | 少见 |
| D群 | 宋内志贺菌 | **症状最轻**，多为不典型轻型，易引起食物型暴发 |

**速记**：A最毒、B最常见且最爱慢性、D最轻

### 4.2 流行病学

- **传染源**：患者和带菌者（慢性带菌者是重要传染源）
- **传播途径**：**粪-口传播**（污染的水、食物、手、苍蝇）
- 流行季节：**夏秋季**
- 人群普遍易感，感染后免疫力不持久，可反复感染
- 感染菌量很低即可致病（10~100个菌）——传染性强

### 4.3 病理

- **病变部位**：主要在**结肠**，以**乙状结肠和直肠**最重
- 急性期：黏膜充血水肿→浅表溃疡→脓性渗出
- 慢性期：黏膜增厚、肠壁增厚、溃疡修复与再发交替
- 中毒型菌痢：肠道病变反而**轻**（因为主要是内毒素介导的全身反应）

### 4.4 临床分型 ⭐

#### ① 急性普通型（典型）——最常见

- 起病急，畏寒发热（38~39°C）
- 腹痛：阵发性，以**左下腹**为主（乙状结肠区）
- 腹泻：初为稀水便 → 迅速转为**黏液脓血便**
  - 量少、次数多（每日10~20次甚至更多）
  - 有腥臭味
- **里急后重（tenesmus）**⭐：频繁便意+排便不尽感+肛门坠胀，排出量少
  - 这是菌痢**最具特征性**的症状
  - 机制：炎症刺激直肠壁内感受器→排便反射亢进
- 查体：左下腹压痛，肠鸣音亢进

#### ② 急性轻型（非典型）

- 全身中毒症状轻，可无发热或低热
- 腹泻次数少，可无脓血便，仅稀便或黏液便
- 里急后重不明显
- 易误诊为急性肠炎、误漏诊 → 成为隐性传染源

#### ③ 急性中毒型 ⭐⭐⭐——考试最爱出

**核心考点**：好发于 **2~7岁** 健康儿童（营养状况好的胖小孩也会得！）

**最大陷阱**：**全身中毒症状极重，但肠道症状可不明显甚至缺如**
- 起病极急，数小时内进展
- 高热（40°C以上）、惊厥、意识障碍先于腹泻出现
- 此时粪便可能还是正常的——容易误诊！

**确诊方法**：**直肠拭子或盐水灌肠**取标本（不能等自行排便）→ 镜检大量脓细胞+培养出志贺菌

三个亚型：

| 亚型 | 核心表现 | 机制 |
|------|----------|------|
| **休克型（周围循环衰竭型）** | 面色苍灰、四肢厥冷、皮肤花纹、血压下降、脉细速、少尿 | 内毒素→微循环障碍→感染性休克 |
| **脑型（呼吸衰竭型）** | 剧烈头痛、反复惊厥、昏迷 → **脑疝** → 呼吸衰竭 | 内毒素→脑微循环障碍→脑水肿→颅内高压 |
| **混合型** | 上述两型同时或先后出现 | **预后最差，病死率最高** |

**脑型的致死机制**：脑水肿 → 颅内压增高 → 脑疝形成 → 呼吸中枢受压 → 中枢性呼吸衰竭 → 死亡

#### ④ 慢性菌痢

- 定义：病程 **> 2个月**不愈
- 最常由**福氏志贺菌（B群）**引起
- 三种表现形式：

| 类型 | 特点 |
|------|------|
| 慢性迁延型 | 症状时轻时重，长期不愈 |
| 急性发作型 | 有慢性菌痢史，因进食不当/受凉等诱发急性发作 |
| 隐匿型 | 无明显症状，但粪便培养阳性，**最危险的传染源** |

### 4.5 实验室检查

| 检查 | 结果 | 意义 |
|------|------|------|
| 血常规 | WBC升高，中性粒细胞为主 | 细菌感染证据 |
| **粪便常规** | **大量WBC（脓细胞）+红细胞**，可见巨噬细胞 | 初步诊断依据 |
| **粪便培养** | 分离出志贺菌 | **确诊金标准** |
| 乙状结肠镜 | 黏膜弥漫充血水肿、浅表溃疡、脓性分泌物附着 | 慢性菌痢辅助诊断 |

### 4.6 鉴别诊断 ⭐

| 鉴别疾病 | 与菌痢的区别要点 |
|----------|-----------------|
| **阿米巴痢疾** | 起病较缓，中毒症状轻，腹痛轻，里急后重**不明显**；大便为**暗红色果酱样**，量多有腥臭；镜检找到**溶组织内阿米巴滋养体**（含吞噬红细胞）；病变以**盲肠和升结肠**为主（菌痢在乙状结肠直肠）；溃疡为**口小底大烧瓶样**（菌痢为浅表溃疡） |
| **急性肠炎** | 水样便为主，无脓血便，无里急后重，粪便培养志贺菌阴性 |
| **肠套叠（小儿）** | 阵发性哭闹（腹痛），果酱样血便，腹部可扪及腊肠样包块，B超可确诊 |
| **直肠癌** | 中老年，便血+排便习惯改变，直肠指检可触及肿物 |

**菌痢 vs 阿米巴痢疾速记对比**：

| 项目 | 细菌性痢疾 | 阿米巴痢疾 |
|------|-----------|-----------|
| 起病 | 急 | 缓 |
| 全身中毒症状 | 重 | 轻 |
| 里急后重 | **显著** | 不明显 |
| 腹痛部位 | 左下腹 | 右下腹（盲肠区） |
| 大便性状 | 黏液**脓**血便，量少 | **暗红色果酱样**，量多 |
| 大便镜检 | 大量脓细胞+红细胞 | **阿米巴滋养体**（吞噬红细胞），脓细胞少 |
| 病变部位 | **乙状结肠、直肠** | **盲肠、升结肠** |
| 溃疡形态 | 浅表、弥漫 | **口小底大（烧瓶样）** |

### 4.7 治疗

#### 急性普通型
- **消化道隔离**至症状消失、粪便培养2次阴性
- 补液：纠正脱水和电解质紊乱
- **抗菌治疗**：
  - 成人首选：**喹诺酮类**（左氧氟沙星、环丙沙星）
  - 儿童（喹诺酮禁用）：**头孢类**（头孢曲松、头孢克肟）
  - 备选：阿奇霉素、磷霉素
  - 疗程：**5~7天**
- 注意：**不宜使用止泻药**（如洛哌丁胺），因为会延缓毒素和细菌排出

#### 急性中毒型——急救重点 ⭐

**通用处理**：
- 抗菌：静脉用药（头孢类/喹诺酮类）
- 降温：物理+药物
- 直肠拭子/盐水灌肠留取粪便标本

**休克型**：
- 扩容：快速补液（生理盐水、乳酸林格）
- 纠酸：5% NaHCO₃
- 血管活性药：多巴胺/去甲肾上腺素
- 必要时糖皮质激素（短程）
- 处理原则同感染性休克

**脑型**：
- **20%甘露醇**快速静滴降颅压（防脑疝！）
- 止惊：地西泮静推
- 吸氧，保持气道通畅
- 地塞米松减轻脑水肿
- 呼吸衰竭时气管插管+机械通气

**混合型**：兼顾抗休克+降颅压，预后最差

#### 慢性菌痢
- 根据药敏选抗菌药物，疗程延长至**10~14天**
- 联合用药
- 寻找并治疗可能的诱因（肠道功能紊乱、营养不良等）
- 保留灌肠：可用抗生素保留灌肠辅助治疗

### 4.8 预防

- 管理传染源：早发现、早隔离、早治疗；彻底治疗慢性患者和带菌者
- 切断传播途径：注意饮食饮水卫生、消灭苍蝇、饭前便后洗手
- 保护易感人群：目前无常规疫苗推荐

## 五、细菌性食物中毒 Bacterial Food Poisoning

> 考试重点：各病原的"标签特征"对比 + 潜伏期 + 毒素型vs感染型区分

### 5.1 分类

- **毒素型**：细菌在食物中产生毒素，人吃下毒素后发病（潜伏期短，起病急）
- **感染型**：活菌随食物进入肠道，在体内繁殖致病（潜伏期相对长）

### 5.2 各病原高频考点对比表 ⭐⭐⭐

| 病原 | 分类 | 关联食物 | 潜伏期 | 核心特征（考试标签） |
|------|------|----------|--------|---------------------|
| **金黄色葡萄球菌** | 毒素型（肠毒素） | 剩饭、乳制品、糕点 | **1~6h（最短）** | **呕吐为主**，腹泻轻；体温正常或低热；肠毒素**耐热**（100°C 30min不灭活），加热食物不能预防 |
| **沙门菌** | 感染型 | 蛋类、肉类、乳类 | 6~48h | **最常见**的细菌性食物中毒；发热+腹痛+腹泻（黄绿色水样便），可伴菌血症 |
| **副溶血弧菌** | 感染型（+肠毒素） | **海产品**（生食） | 6~24h | 沿海地区多见；腹痛剧烈，水样便或血水样便 |
| **肉毒杆菌** | 毒素型（神经毒素） | **自制发酵食品**（豆酱、臭豆腐、罐头） | 12~48h | **无消化道症状**！以**神经麻痹**为主：对称性下行性弛缓性瘫痪（眼肌→咽肌→呼吸肌），瞳孔散大，意识清楚；死因：呼吸肌麻痹 |
| 产气荚膜梭菌 | 毒素型（肠毒素） | 肉类 | 8~24h | 水样腹泻为主，呕吐少见 |
| 大肠杆菌 O157:H7 | 感染型（+志贺毒素） | 牛肉、生菜 | 3~9天 | 出血性肠炎；可并发**HUS（溶血尿毒综合征）**——小儿急性肾衰最常见原因之一 |

### 5.3 治疗原则

| 处理 | 要点 |
|------|------|
| 通用 | 催吐/洗胃（早期）、补液纠正脱水、对症 |
| 感染型 | 可用抗生素（喹诺酮类、头孢类） |
| 金葡菌毒素型 | 一般不需抗生素，补液对症即可，病程短自限 |
| **肉毒中毒** | **多价抗毒素血清**——特效治疗⭐；早期洗胃；呼吸肌麻痹时机械通气；**不用抗生素**（毒素已形成，杀菌无意义） |

### 5.4 考试易混点

- 题目说"聚餐后数小时多人呕吐，不发热" → **金葡菌**
- 题目说"吃自制豆酱后视物模糊、吞咽困难、对称性瘫痪" → **肉毒中毒**
- 题目说"吃海鲜后腹痛腹泻" → **副溶血弧菌**
- 题目说"吃蛋类/肉类后发热+腹泻黄绿色水样便" → **沙门菌**
- 肉毒杆菌产的毒素是已知**最强的生物毒素**，但**不耐热**（80°C 30min灭活）——与金葡菌肠毒素耐热相反

## 六、结核病 Tuberculosis

### 6.1 病原学

- **结核分枝杆菌（MTB, Mycobacterium tuberculosis）**
- **抗酸染色阳性**（齐-尼染色法，红色）——分枝杆菌的标志性特征
- **专性需氧菌**——肺尖部氧分压最高，所以继发性结核好发于肺尖
- 生长极慢（倍增时间15~20h）→ 培养需2~8周
- 细胞壁富含脂质 → 抗酸性、耐干燥、耐多数消毒剂
- **对紫外线敏感**（日光消毒有效）
- 不产内/外毒素，致病靠**菌体成分引发的免疫损伤**

### 6.2 流行病学

| 要素 | 内容 |
|------|------|
| 传染源 | **痰涂片阳性的肺结核患者**（排菌者），涂阴培阳者传染性较低 |
| 传播途径 | **呼吸道飞沫传播**（飞沫核，可长时间悬浮） |
| 易感因素 | HIV感染（最重要）、糖尿病、免疫抑制剂、营养不良、矽肺、老年、婴幼儿 |
| 感染后结局 | 90%为潜伏感染（LTBI），仅约**5~10%**发展为活动性结核 |

### 6.3 发病机制与病理

**核心病理改变**：结核性肉芽肿（结核结节）
- 中心：**干酪样坏死**
- 周围：**上皮样细胞 + 朗汉斯巨细胞（Langhans giant cell）**
- 外层：淋巴细胞+纤维组织

**免疫机制**：
- **迟发型超敏反应（IV型变态反应）**——是结核免疫反应的核心
- 变态反应（组织损伤）与免疫保护共存、相互伴随
- 这就是为什么PPD阳性说明既有免疫力又有组织损伤的潜力

### 6.4 肺结核分类——考试重点 ⭐⭐

#### ① 原发性肺结核（Primary TB）

| 项目 | 内容 |
|------|------|
| 人群 | **儿童**、初次感染者 |
| 原发灶位置 | **肺中下野**（与继发性相反！）——上叶下部、下叶上部 |
| 典型表现 | **原发综合征**：原发灶 + 引流淋巴管炎 + 肺门/纵隔淋巴结肿大（X线呈"哑铃状"或"双极现象"） |
| 预后 | 多数可自愈，少数可血行播散 |
| 并发症 | 支气管淋巴结结核（压迫支气管→肺不张）、血行播散性结核、结核性脑膜炎 |

#### ② 血行播散性肺结核

**急性粟粒性肺结核**⭐：
- 大量MTB一次性入血 → 双肺弥漫性粟粒样阴影
- 影像学特征：**"三均匀"——大小均匀、密度均匀、分布均匀**
- 粟粒大小：1~2mm
- 临床表现：急性高热、全身中毒症状重，可伴结核性脑膜炎
- 注意：**起病2周内X线可能正常**，需复查

亚急性/慢性血行播散：
- 少量多次入血
- 影像：大小不一、密度不均、分布不均（"三不均"）

#### ③ 继发性肺结核（Post-primary TB）——最常见、考试最多 ⭐⭐⭐

**好发部位**：**上叶尖后段、下叶背段**（氧分压最高处）

| 类型 | 特点 | 考试关键词 |
|------|------|-----------|
| **浸润性肺结核** | **最常见的类型**，锁骨上下区斑片状浸润 ± 空洞 | 最常见 |
| **空洞性肺结核** | 干酪样坏死液化排出→空洞形成，空洞壁薄 | **痰菌阳性率最高**，传染性最强 |
| **结核球（结核瘤）** | 圆形/类圆形病灶，直径2~4cm，纤维包裹内含干酪坏死 | 需与肺癌鉴别 |
| **干酪样肺炎** | 大片干酪样坏死，全身中毒症状极重 | 急性重症，类似肺炎 |
| **纤维空洞性肺结核** | 厚壁空洞+肺纤维化+支气管播散灶，病程长 | **"老结核"**，传染性最强，难治 |

**记忆技巧**：传染性排序 = 空洞型 ≈ 纤维空洞型 > 浸润型 > 结核球（封闭最弱）

### 6.5 临床表现

**全身症状**：
- **低热**：午后低热为著（典型）
- **盗汗**：入睡后出汗，醒后汗止
- 乏力、纳差、消瘦
- 女性可有月经不调

**呼吸道症状**：
- **咳嗽**：最常见的症状，干咳或少量白痰
- **咯血**⭐：
  - 小量咯血最常见（痰中带血）
  - 中~大量咯血：空洞内**Rasmussen动脉瘤**破裂
  - **咯血最危险的并发症不是失血而是窒息**——血块堵塞气道
  - 大咯血窒息的急救：头低脚高体位+拍背+保持气道通畅
- 胸痛：累及胸膜时
- 呼吸困难：广泛病变或胸腔积液时

### 6.6 诊断 ⭐⭐

#### 病原学检查

| 检查 | 特点 |
|------|------|
| **痰涂片抗酸染色** | 快速（数小时），但**敏感性低（～50%）**；涂阳即可确诊活动性TB |
| **痰培养** | **金标准**，可做药敏；缺点：耗时长（固体2~8周，液体1~2周） |
| **GeneXpert MTB/RIF** | 分子生物学方法，**2小时出结果**，同时检测MTB和利福平耐药；WHO推荐初筛工具⭐ |

#### 免疫学检查

| 检查 | PPD/TST（结核菌素试验） | T-SPOT/IGRA（γ-干扰素释放试验） |
|------|------------------------|-------------------------------|
| 原理 | 皮内注射结核菌素，48~72h看硬结 | 检测外周血T细胞对MTB特异抗原的IFN-γ释放 |
| 判读 | 硬结≥5mm阳性（HIV/免疫缺陷）；≥10mm阳性（一般人群）；≥15mm强阳性 | 定量报告 |
| 受BCG影响 | **受影响**（接种BCG后也会阳性） | **不受影响**⭐——最大优势 |
| 假阴性 | 免疫抑制、重症TB、麻疹后、老年、营养不良 | 免疫抑制时也可假阴性 |
| 临床意义 | 不能区分感染和接种，强阳性提示活动性TB可能 | 更特异，适合区分LTBI和BCG接种 |

**考试陷阱**：PPD阴性**不能排除**结核（假阴性多种原因），PPD阳性也**不能确诊**结核（BCG接种后也阳性）

#### 影像学特征

| 类型 | X线/CT表现 |
|------|-----------|
| 原发综合征 | 中下肺野原发灶 + 肺门淋巴结肿大（哑铃状） |
| 浸润性 | 上叶尖后段斑片状浸润 ± 空洞 |
| 粟粒性 | 双肺弥漫粟粒阴影，三均匀 |
| 空洞 | 薄壁空洞，内壁光滑（与癌性空洞的厚壁偏心鉴别） |
| 结核球 | 圆形致密影，2~4cm，周围可有卫星灶 |
| 纤维空洞型 | 厚壁空洞+纤维条索+肺容积缩小+对侧播散灶 |

**结核空洞 vs 癌性空洞鉴别**：

| 特征 | 结核空洞 | 癌性空洞 |
|------|----------|----------|
| 壁厚 | 薄壁 | **厚壁** |
| 内壁 | 光滑 | **不规则、结节状** |
| 位置 | 上叶 | 不定 |
| 周围 | 卫星灶、纤维索条 | 分叶征、毛刺征 |
| 偏心 | 多居中 | **偏心性**多见 |

### 6.7 治疗 ⭐⭐⭐

#### 化疗原则（五字诀，必背）

**早期、联合、适量、规律、全程**

#### 标准方案

**初治肺结核**：**2HRZE / 4HR**

| 阶段 | 时间 | 药物 | 含义 |
|------|------|------|------|
| 强化期 | **2个月** | H + R + Z + E（四联） | 迅速杀菌、减少排菌量 |
| 巩固期 | **4个月** | H + R（两联） | 消灭残余菌、防复发 |

总疗程 = **6个月**

**复治/耐药/重症**：延长疗程，可加链霉素(S)，具体方案依药敏

#### 抗结核药物毒性——必考 ⭐⭐⭐

| 药物 | 代号 | 核心作用 | **关键毒性** | 备注 |
|------|------|----------|-------------|------|
| 异烟肼 (Isoniazid) | **H** | 早期杀菌力**最强** | **周围神经炎**（补VitB₆预防）；肝毒性 | 全杀菌药（细胞内外均有效） |
| 利福平 (Rifampicin) | **R** | 杀菌+灭菌（杀持留菌） | 肝毒性；**体液橘红色**（尿、泪、汗）；**肝药酶强诱导剂**（CYP3A4）→加速多种药物代谢 | 药物相互作用最多的抗结核药 |
| 吡嗪酰胺 (Pyrazinamide) | **Z** | 杀酸性环境中的菌（巨噬细胞内） | **高尿酸血症/痛风**⭐；肝毒性（**肝毒性最强**的抗结核药） | 仅在强化期使用 |
| 乙胺丁醇 (Ethambutol) | **E** | 抑菌 | **视神经炎**（视力下降、红绿色觉异常）——儿童不推荐使用（不能可靠报告视力变化） | 定期查视力+色觉 |
| 链霉素 (Streptomycin) | **S** | 杀菌（细胞外） | **耳毒性（第Ⅷ颅神经损害）**：前庭功能障碍（眩晕）+耳蜗损害（耳聋）；肾毒性 | **孕妇禁用**（致胎儿Ⅷ颅神经损害） |

**毒性速记口诀**：
- 异烟肼伤**神经**（补B₆）
- 利福平变**橘红**（酶诱导）
- 吡嗪酰胺伤**肝+尿酸**
- 乙胺丁醇伤**眼**
- 链霉素伤**耳+肾**

#### 利福平的药物相互作用——放疗科特别注意 ⭐

利福平是**CYP3A4强诱导剂**，加速以下药物代谢（血药浓度降低）：
- 华法林（抗凝效果减弱）
- 口服避孕药（避孕失败）
- 地塞米松/泼尼松（糖皮质激素效果减弱）
- **部分TKI靶向药**（如厄洛替尼、吉非替尼等→疗效降低）
- 环孢素、他克莫司（免疫抑制效果减弱）
- 美沙酮、地西泮等

→ 合并用利福平时，需评估并调整合用药物的剂量或改用利福布汀(Rifabutin，酶诱导作用弱)

#### 耐药结核

| 类型 | 定义 |
|------|------|
| **MDR-TB（多耐药结核）** | 至少对 **H + R** 同时耐药 |
| **XDR-TB（广泛耐药结核）** | MDR基础上 + 对**氟喹诺酮类** + 至少一种注射类二线药耐药 |

- GeneXpert可快速检测利福平耐药 → 筛查MDR-TB
- 治疗：需要二线药物方案，疗程 ≥ 18~20个月

### 6.8 预防

#### BCG疫苗（卡介苗）

| 项目 | 内容 |
|------|------|
| 接种时间 | **出生时**（新生儿常规接种） |
| 疫苗类型 | **减毒活疫苗** |
| 保护作用 | 预防**严重结核病**（粟粒性结核、结核性脑膜炎），尤其是儿童 |
| 局限性 | **不能预防MTB感染**，不能预防成人肺结核，保护力有限且随年龄下降 |
| 禁忌 | 免疫缺陷（HIV感染者）、活动性结核 |

#### 预防性治疗（化学预防）

适应证：潜伏结核感染（LTBI）高危者
- HIV + LTBI
- 密切接触活动性TB + PPD/IGRA阳性
- 免疫抑制剂使用前LTBI阳性
- PPD近期转阳者

方案：
- 异烟肼(H) **6~9个月**——经典方案
- 利福平(R) **4个月**——替代方案
- 异烟肼+利福喷汀 每周1次×3个月（3HP方案）——依从性好

### 6.9 与放疗科交叉考点 ⭐

| 场景 | 要点 |
|------|------|
| 放化疗后免疫抑制 | LTBI活化/再燃风险增加；高危患者需筛查+预防性治疗 |
| 利福平与肿瘤药物相互作用 | 利福平加速TKI、激素等代谢→疗效降低；考虑换用利福布汀或调整方案 |
| 肺部肿块鉴别 | 结核球 vs 肺癌（CT特征+PET-CT+穿刺活检） |
| 胸腔积液鉴别 | 结核性胸膜炎（淋巴细胞为主、ADA升高、蛋白高）vs 恶性胸腔积液（细胞学找到癌细胞） |
| 放疗前筛查 | 胸部放疗前需评估有无活动性TB或LTBI，必要时先抗结核 |
| 麻疹后PPD转阴 | 见第二章麻疹，免疫抑制可致PPD假阴性——同理放化疗后也会 |

## 七、败血症与感染性休克 Septicemia & Septic Shock

> 这两个本质上是同一条病理生理链的不同阶段，放在一起理解更顺畅。

### 7.1 概念辨析——先把名词理清 ⭐

| 概念 | 定义 | 关键词 |
|------|------|--------|
| **菌血症 (Bacteremia)** | 细菌一过性入血，血培养阳性，可无明显症状 | 一过性、可无症状 |
| **败血症 (Septicemia)** | 病原菌侵入血流并**在血中繁殖**，产生毒素，引起全身严重感染症状 | 血中繁殖+全身症状 |
| **脓毒血症 (Pyemia)** | 败血症基础上，细菌栓子随血流到达各脏器，形成**迁徙性脓肿** | 败血症+多发脓肿 |
| **脓毒症 (Sepsis)** | 感染引起的**失调的宿主反应**导致危及生命的器官功能障碍（Sepsis-3定义） | 感染+器官功能障碍 |
| **感染性休克 (Septic Shock)** | 脓毒症亚型，经充分液体复苏后仍需血管活性药维持MAP≥65mmHg，且乳酸>2mmol/L | 脓毒症+顽固低血压+高乳酸 |

**Sepsis-3诊断工具**：**SOFA评分**（序贯器官功能衰竭评估），较基线升高≥2分即考虑脓毒症。快速筛查用**qSOFA**（呼吸≥22、意识改变、收缩压≤100mmHg，≥2项阳性需警惕）。

### 7.2 败血症

#### 7.2.1 常见病原菌

| 分类 | 常见菌种 | 临床特点 |
|------|----------|----------|
| **G⁺球菌** | 金黄色葡萄球菌、表皮葡萄球菌、肠球菌、链球菌 | 迁徙性脓肿多见（皮肤、肺、肝、脑）；皮疹多见；金葡菌→急性感染性心内膜炎 |
| **G⁻杆菌** | 大肠埃希菌、肺炎克雷伯、铜绿假单胞、鲍曼不动杆菌 | **更易引起感染性休克**（内毒素→炎症级联）；常继发于泌尿道、胆道、腹腔感染 |
| 真菌 | 白色念珠菌 | 长期广谱抗生素/免疫抑制后 |

**常见入侵途径**（题目常给线索）：
- 皮肤伤口/烧伤 → 金葡菌
- 泌尿道操作/尿路感染 → G⁻杆菌（大肠埃希菌）
- 胆道感染 → G⁻杆菌
- 静脉导管 → 表皮葡萄球菌、金葡菌（CRBSI）
- 腹腔感染 → G⁻杆菌+厌氧菌混合

#### 7.2.2 临床表现

**三大核心表现**：
1. **毒血症状**：寒战、高热（弛张热或间歇热），大汗
   - G⁺菌：持续高热多见
   - G⁻菌：间歇寒战高热（与内毒素释放周期相关）
2. **皮疹**：
   - 瘀点瘀斑 → 脑膜炎球菌败血症（暴发性流脑）
   - 脓疱疹 → 金葡菌
   - 猩红热样皮疹 → 链球菌
3. **肝脾大**：尤其是脾大（单核-巨噬细胞系统反应性增生）

**其他**：关节痛、迁徙性病灶（脓肿）、感染性心内膜炎征象

#### 7.2.3 实验室检查

| 检查 | 要点 |
|------|------|
| **血培养** | **金标准**⭐；寒战高热时采血，抗生素使用前采集；至少采2~3套（需氧+厌氧），不同部位 |
| 血常规 | WBC升高+中性粒升高+核左移；重症可WBC降低（提示预后差） |
| PCT（降钙素原） | 细菌感染标志物，>0.5ng/mL高度提示细菌感染，可指导抗生素疗程 |
| CRP | 升高，但特异性不如PCT |
| 乳酸 | >2mmol/L提示组织灌注不足 |

#### 7.2.4 治疗原则

**抗生素使用原则**（必考）：
1. **尽早使用**：诊断后**1小时内**启动经验性抗菌治疗（每延迟1小时死亡率增加）
2. **先经验后目标**：采血培养 → 立即经验性用药 → 药敏结果回报后调整
3. **足量、静脉给药**
4. **联合用药**：重症初始可联合，药敏明确后降阶梯
5. **疗程充分**：体温正常后7~10天，或血培养转阴后继续用药

**经验性选药思路**：

| 感染来源 | 经验性抗菌方案 |
|----------|---------------|
| 社区获得性/来源不明 | 三代头孢（头孢曲松/头孢噻肟）± 甲硝唑 |
| 泌尿道来源 | 喹诺酮类 或 三代头孢 |
| 腹腔来源 | 三代头孢 + 甲硝唑（覆盖厌氧菌） |
| 导管相关（CRBSI） | 万古霉素（覆盖MRSA）± 抗G⁻药物；拔除/更换导管 |
| 院内感染/ICU | 碳青霉烯类（亚胺培南/美罗培南）± 万古霉素 |
| 粒缺伴发热 | 抗假单胞β-内酰胺（头孢吡肟/哌拉西林他唑巴坦/碳青霉烯） |

---

### 7.3 感染性休克

#### 7.3.1 病理生理链

```
感染 → 病原体/毒素激活免疫系统 → 炎症因子大量释放（TNF-α、IL-1、IL-6等）
→ 全身炎症反应失控 → 血管扩张 + 毛细血管通透性增加 + 心肌抑制
→ 有效循环血量不足 → 组织灌注不足 → 器官功能障碍 → 多器官功能衰竭(MODS)
```

**核心机制**：分布性休克（血管扩张+容量再分配），与低血容量性休克和心源性休克鉴别

#### 7.3.2 暖休克 vs 冷休克——经典考点 ⭐

| 特征 | 暖休克（早期/高动力型） | 冷休克（晚期/低动力型） |
|------|----------------------|----------------------|
| 阶段 | 休克代偿期（早期） | 休克失代偿期（晚期） |
| 皮肤 | **温暖、干燥、潮红** | **湿冷、苍白、花斑** |
| 心输出量 | 增高或正常 | **降低** |
| 外周血管阻力 | **降低**（血管扩张） | 增高（代偿性收缩） |
| 脉搏 | 洪大有力 | 细速 |
| 血压 | 收缩压可正常，脉压增大 | **血压下降** |
| 意识 | 烦躁或正常 | 淡漠、嗜睡、昏迷 |
| 尿量 | 减少 | **明显减少或无尿** |
| 病原 | **G⁻杆菌多见** | G⁺或晚期均可 |

**考试技巧**：题目描述"感染后四肢温暖、面色潮红、脉压增大" → 暖休克/感染性休克早期

#### 7.3.3 诊断要点

脓毒症基础上出现：
- 经**充分液体复苏**后仍存在低血压，需要**血管活性药物**维持MAP ≥ 65 mmHg
- **血清乳酸 > 2 mmol/L**

#### 7.3.4 治疗——SSC核心策略 ⭐⭐⭐

**黄金1小时Bundle（Hour-1 Bundle）**：

| 措施 | 具体内容 |
|------|----------|
| ① 测乳酸 | 乳酸>2mmol/L需复测以评估复苏效果 |
| ② 血培养 | 抗生素使用**前**留取血培养 |
| ③ 抗生素 | 识别脓毒症**1小时内**给予广谱抗菌药物 |
| ④ 液体复苏 | 低血压或乳酸≥4mmol/L → **30ml/kg晶体液**（生理盐水或乳酸林格），3小时内完成 |
| ⑤ 血管活性药 | 液体复苏后仍低血压 → **去甲肾上腺素（首选）**⭐，目标MAP≥65mmHg |

**液体复苏要点**：
- 首选**晶体液**（生理盐水或平衡盐液）
- 初始30ml/kg，之后根据容量反应性调整
- 评估容量反应性：被动抬腿试验(PLR)、每搏量变异度(SVV)等
- 避免过度补液（液体过负荷加重肺水肿）

**血管活性药物**：

| 药物 | 地位 | 作用 |
|------|------|------|
| **去甲肾上腺素 (NE)** | **一线首选**⭐ | α为主+少量β，升压效果好，对心率影响小 |
| 肾上腺素 | 二线，NE不足时加用 | α+β，升压+增强心肌收缩 |
| 血管加压素 | 二线辅助 | 非儿茶酚胺类，NE效果不佳时联合（≤0.03U/min） |
| 多巴胺 | 快速性心律失常低风险患者可选 | 低剂量扩肾血管（存争议）；β作用增加心输出量；高剂量α缩血管 |
| 多巴酚丁胺 | 心功能不全时辅助 | β1为主，增强心肌收缩力 |

**糖皮质激素**：
- 适应证：充分液体复苏+血管活性药后血压仍不稳定（**难治性休克**）
- 用法：**氢化可的松 200mg/d**，持续静滴或分次给药
- 血流动力学稳定后尽快减停

**其他治疗**：
- 血糖控制：目标 ≤ 10 mmol/L（避免低血糖）
- 预防应激性溃疡：PPI
- 预防深静脉血栓：低分子肝素
- 器官功能支持：机械通气（ARDS时肺保护性通气）、CRRT（急性肾损伤时）

### 7.4 与放疗科交叉考点 ⭐

**粒细胞缺乏伴发热（Febrile Neutropenia, FN）**——化疗后最凶险的急症之一：

| 项目 | 内容 |
|------|------|
| 定义 | ANC（中性粒细胞绝对值）< 0.5×10⁹/L + 体温≥38.3°C单次或≥38.0°C持续1h |
| 风险 | 极易进展为败血症→感染性休克，病死率高 |
| 经验性抗菌 | **立即**给予抗假单胞菌β-内酰胺类：头孢吡肟/哌拉西林他唑巴坦/碳青霉烯（选一） |
| 加万古霉素指征 | 导管相关感染疑似、皮肤软组织感染、MRSA定植、血流动力学不稳定 |
| 抗真菌 | 广谱抗菌4~7天仍发热 → 经验性加抗真菌（卡泊芬净/两性霉素B脂质体） |
| G-CSF | 预期粒缺持续>7天或已有严重感染时考虑使用 |

**放疗相关败血症注意**：
- 盆腔放疗后放射性肠炎 → 肠黏膜屏障破坏 → 肠源性菌血症（G⁻杆菌为主）
- 头颈部放疗后口腔黏膜炎 → 口腔菌群入血
- 同步放化疗期间骨髓抑制 → FN风险叠加

## 八、流行性出血热 Hemorrhagic Fever with Renal Syndrome (HFRS)

> 五期经过是本病的灵魂，考试围绕各期特征和治疗反复出题。

### 8.1 病原学

- **汉坦病毒（Hantavirus）**，布尼亚病毒科，有包膜，负链RNA
- 两种主要血清型（考试会让你配对）：

| 血清型 | 宿主（传染源） | 类型 | 病情严重程度 |
|--------|---------------|------|-------------|
| **汉滩病毒 (Hantaan)** | **黑线姬鼠**（野鼠） | 野鼠型 | **重型** |
| **首尔病毒 (Seoul)** | **褐家鼠**（家鼠） | 家鼠型 | **轻型** |

**速记**：野鼠在野外 → 汉滩 → 病重；家鼠在家里 → 首尔 → 病轻

### 8.2 流行病学

| 要素 | 内容 |
|------|------|
| 传染源 | **鼠类**（排泄物中含大量病毒） |
| 传播途径 | ① 呼吸道（吸入含病毒鼠排泄物气溶胶）——主要途径；② 消化道（污染食物）；③ 接触传播（破损皮肤黏膜）；④ 虫媒（螨叮咬）；⑤ 垂直传播（罕见） |
| 流行季节 | 野鼠型：**秋冬季**（11~1月）；家鼠型：**春季**（3~5月） |
| 好发人群 | 青壮年男性、农村野外作业者 |
| 感染后免疫 | 持久免疫，很少二次感染 |

### 8.3 发病机制（理解即可）

```
病毒入血 → 全身小血管和毛细血管内皮损伤
→ 血管通透性↑ → 血浆外渗 → 组织水肿 + 有效循环血量↓
→ 同时启动DIC → 消耗性凝血障碍 → 出血
→ 肾脏损伤最突出（肾髓质充血、出血、坏死）
```

核心病理：**全身小血管损伤** → 三大基本病变：**血浆外渗、出血、肾损害**

### 8.4 临床表现——五期经过 ⭐⭐⭐（必考中的必考）

```
发热期 → 低血压休克期 → 少尿期 → 多尿期 → 恢复期
```

**总原则**：前三期可重叠（重型），轻型可越过休克期和/或少尿期直接进入多尿期

---

#### ① 发热期（3~7天）

**三大特征群**（考试最爱考的组合题）：

**🔴 "三痛"**：**头痛、腰痛、眼眶痛**
- 头痛剧烈
- **腰痛**：肾周水肿和充血所致（肾损害的早期信号）
- 眼眶痛：球后组织水肿

**🔴 "三红"**：**颜面、颈部、上胸部皮肤潮红**
- 称为**"醉酒貌"**（酒醉面容）⭐
- 机制：皮肤小血管扩张充血

**🔴 充血出血表现**：
- **球结膜充血、水肿**（有时呈片状出血）——非常有辨识度的体征
- **软腭出血点**
- **腋下出血点**（搓衣样分布）
- 皮肤瘀点瘀斑
- 注射部位可有大片瘀斑

**肾损害开始出现**：
- 蛋白尿（出现早、量大）
- 尿中可见膜状物（蛋白+脱落细胞管型）

**发热期速记口诀**：三痛三红醉酒貌，球结膜充血是信号，蛋白尿早出现，腋下瘀点搓衣找

---

#### ② 低血压休克期（约第4~6天）

**最关键的考点**：**"热退病重"**⭐⭐
- 体温下降时病情反而加重（与多数传染病热退病减相反）
- 血压下降甚至休克
- 机制：血浆大量外渗 → 有效循环血量急剧减少

| 表现 | 具体内容 |
|------|----------|
| 血压 | 下降，脉压缩小 |
| 全身 | 面色苍白、四肢厥冷、出冷汗 |
| 心率 | 加快 |
| 尿量 | 开始减少 |
| 持续时间 | 数小时 ~ 3天 |

**注意**：此期可与发热期重叠（热未退血压已降），也可与少尿期重叠

---

#### ③ 少尿期（约第5~8天）——**最危险的阶段，死亡率最高** ⭐⭐⭐

- 少尿定义：尿量 < 400 ml/d；无尿：< 100 ml/d
- 持续2~5天（短则1天，长则2周以上）

**四大威胁**：

| 威胁 | 内容 |
|------|------|
| **尿毒症** | BUN/Cr显著升高→恶心呕吐、意识障碍、尿毒症脑病 |
| **高血容量综合征**⭐ | 血浆回渗（外渗的血浆回流至血管内）→ 血容量增加 → **血压升高、脉压增大、头痛加重、心衰、肺水肿**——与休克期的低血压形成鲜明反转 |
| **出血** | DIC+血小板减少 → 皮肤瘀斑、消化道出血、肺出血、颅内出血 |
| **电解质/酸碱紊乱** | **高钾血症**（最危险，可致心脏骤停）、低钠、低钙、代谢性酸中毒 |

**"高血容量综合征"是出血热少尿期的特色**——其他肾衰竭不一定有这个特征，因为出血热有前期大量血浆外渗后回渗的病理过程

---

#### ④ 多尿期（约第9~14天）

- 肾小管上皮细胞再生修复，但浓缩功能尚未恢复 → 尿量增多
- 日尿量可达 **3000~10000 ml**

| 阶段 | 尿量 | 注意事项 |
|------|------|----------|
| 移行期 | 400~2000 ml/d | 氮质血症仍在，**仍有死亡风险** |
| 多尿早期 | >2000 ml/d | 仍需警惕感染和电解质紊乱 |
| 多尿后期 | 逐渐下降 | **脱水、低钾血症、低钠血症**——主要危险 |

**考试陷阱**：多尿期≠安全期。移行期和多尿早期患者仍可死亡（氮质血症未纠正+继发感染）

---

#### ⑤ 恢复期

- 尿量恢复正常（1500~2000 ml/d）
- 肾功能逐渐恢复，约需1~3个月
- 少数重型患者可遗留肾功能损害

---

### 8.5 实验室检查——三大特征 ⭐⭐

| 特征 | 内容 | 意义 |
|------|------|------|
| **血小板减少** | 发热期即开始下降，少尿期最低 | 与出血严重程度相关 |
| **蛋白尿** | 出现早、量大，可有膜状物（管型+蛋白） | 反映肾损害，早期诊断价值 |
| **异型淋巴细胞增多** | 外周血涂片 > 10%有诊断价值 | 病毒感染特征（类似传单） |

**其他**：
- 血常规：WBC升高，核左移
- **HCT升高**（血液浓缩）→ 见于发热后期/休克期，是血浆外渗的证据
- BUN/Cr：少尿期显著升高

**确诊**：血清特异性**IgM抗体**阳性或双份IgG ≥ 4倍升高

### 8.6 各期治疗——"三早一就" + 分期处理 ⭐⭐⭐

**总原则**：**早发现、早休息、早治疗、就近治疗**

| 分期 | 治疗原则 | 关键措施 |
|------|----------|----------|
| **发热期** | 抗病毒+减轻外渗+对症 | **利巴韦林**（发病4天内用效果最好）；适量补液（维持血容量）；退热以物理降温为主（慎用强退热药防虚脱加速休克） |
| **低血压休克期** | **扩容是核心**⭐ | 补液原则：**早期、快速、适量**；晶体液+胶体液（低右、血浆）；纠酸：5% NaHCO₃；血管活性药：多巴胺（扩容效果不佳时）；注意：过度补液可加重后续高血容量 |
| **少尿期** | "稳、促、导、透"⭐ | **稳**：稳定内环境（纠正高钾、酸中毒）；**促**：呋塞米促进利尿；**导**：甘露醇口服导泻排毒（注意是口服，非静脉降颅压！）；**透**：严重少尿/无尿/高钾/肺水肿 → **血液透析** |
| **多尿期** | 补液补电解质 | 防脱水，防低钾低钠；防继发感染 |
| **恢复期** | 逐步恢复 | 避免过劳，定期复查肾功能 |

**少尿期的"稳促导透"是考试最爱考的治疗口诀**

**少尿期紧急处理优先级**：
1. 高钾血症 → 最先处理（可致心脏停搏）
   - 10%葡萄糖酸钙（拮抗心肌毒性）
   - 胰岛素+葡萄糖（促钾转入细胞内）
   - NaHCO₃（纠酸同时降钾）
   - 离子交换树脂/血液透析
2. 高血容量综合征 → 限液+利尿+必要时放血
3. 尿毒症 → 透析
4. 出血 → 输血小板、新鲜冰冻血浆

### 8.7 预防

- **灭鼠防鼠**——最根本的预防措施⭐
- 个人防护：野外作业时避免接触鼠类及其排泄物
- 食品卫生：防止鼠类污染食物
- 疫苗：双价灭活疫苗（汉滩+首尔型），高危地区人群接种

### 8.8 总结速查卡

```
流行性出血热 = 鼠→人 + 五期经过 + 三痛三红 + 实验室三特征

五期：发热 → 低血压 → 少尿 → 多尿 → 恢复
           ↑            ↑
        热退病重     最危险（稳促导透）

三痛：头痛 + 腰痛 + 眼眶痛
三红：面红 + 颈红 + 胸红 = 醉酒貌

实验室三特征：
  ① 血小板↓
  ② 蛋白尿（早、量大）
  ③ 异型淋巴细胞↑

治疗：三早一就 → 各期对症
  休克期：扩容
  少尿期：稳促导透
  多尿期：补液补钾
```

## 九、伤寒 Typhoid Fever

### 9.1 病原学

- **伤寒沙门菌（Salmonella typhi）**，G⁻杆菌，有鞭毛（周鞭毛，运动活泼）
- 致病因素：**内毒素**（最重要）+ 侵袭力
- 不产外毒素

**抗原结构**（肥达反应的基础）：

| 抗原 | 性质 | 对应抗体 | 临床意义 |
|------|------|----------|----------|
| **O抗原**（菌体抗原） | 脂多糖，耐热 | 抗-O | IgM类，出现早消失快，提示**近期感染** |
| **H抗原**（鞭毛抗原） | 蛋白质，不耐热 | 抗-H | IgG类，出现稍晚持续久，可因**既往感染或预防接种**而升高 |
| **Vi抗原**（荚膜/毒力抗原） | 表面多糖 | 抗-Vi | 与毒力相关；**慢性带菌者检测**的标志物 |

### 9.2 流行病学

| 要素 | 内容 |
|------|------|
| 传染源 | 患者和**慢性带菌者**（带菌>1年，胆囊是主要储菌部位） |
| 传播途径 | **粪-口传播**（水源污染最重要→可致暴发流行） |
| 流行季节 | 夏秋季 |
| 感染后免疫 | 可获得持久免疫 |

### 9.3 发病机制与病理

**发病过程**：
```
口→小肠→肠壁淋巴组织（Peyer斑）→被巨噬细胞吞噬但不被杀灭
→ 经淋巴管 → 肠系膜淋巴结 → 胸导管 → 血流
→ 第一次菌血症（潜伏期末，短暂）
→ 肝、脾、骨髓等单核-巨噬细胞系统内繁殖
→ 第二次菌血症（起病，极期开始，持续、大量）→ 全身各器官受累
```

**"伤寒细胞"**：巨噬细胞吞噬伤寒杆菌、红细胞和细胞碎片后形成的特征性细胞，聚集成**伤寒肉芽肿（伤寒小结）**——病理学特征

**肠道病理变化（按周演进）**⭐——并发症出现时间的基础：

| 病程 | 肠道Peyer斑变化 | 临床对应 |
|------|----------------|----------|
| 第1周 | 增生肿胀 | 初期 |
| 第2周 | **坏死** | 极期 |
| **第3周** | **溃疡形成**⭐ | **并发症高发期（肠出血、肠穿孔）** |
| 第4周 | 愈合 | 恢复期 |

**溃疡特征**：沿肠管**长轴**方向 → 穿孔后不易造成肠腔狭窄（与结核溃疡**垂直于长轴**鉴别）
**好发部位**：**回肠末端**（距回盲瓣约20cm以内）

### 9.4 临床表现（四期）⭐⭐⭐

#### ① 初期（第1周）

- **起病缓慢**（与菌痢的骤起对比——鉴别点）
- 发热：体温呈**阶梯状逐日升高**，约1周内达到39~40°C
- 全身不适、乏力、纳差
- 可有**便秘**（成人，非腹泻！——考试陷阱）
- 也可有腹泻（儿童伤寒腹泻多于便秘）

#### ② 极期（第2~3周）——特征性表现集中爆发 ⭐⭐⭐

**五大特征**（考试核心）：

| 特征 | 具体内容 | 考试要点 |
|------|----------|----------|
| **稽留热** | 体温持续 39~40°C，24h波动 **< 1°C** | 伤寒**最具特征性**的热型 |
| **相对缓脉** ⭐ | 体温升高但脉搏增快**不成比例**（正常每升高1°C脉搏应增加15~20次/min） | 高度提示伤寒；也见于布鲁菌病、军团菌肺炎 |
| **玫瑰疹 (Rose spots)** ⭐ | 淡红色小斑丘疹，直径2~4mm，压之褪色，多见于**胸腹部**，数量 ≤ 20个，病程第7~14天出现，持续2~4天消退，可**分批出现** | 皮疹本身有细菌，可做培养 |
| **肝脾大** | **脾大**更为显著和常见 | 脾大+稽留热+缓脉 → 伤寒经典三联 |
| **表情淡漠** | 神情呆滞、反应迟钝、听力下降、谵妄→"**伤寒面容**" | 中枢神经系统中毒表现 |

**其他**：
- 右下腹轻压痛（回肠末端病变）
- 腹胀、便秘（成人）或腹泻（儿童）
- 无欲面容+相对缓脉+稽留热 = 伤寒的"三板斧"

#### ③ 缓解期（第3~4周）

- 体温波动，逐渐下降
- 症状好转
- **但此时肠道正处于溃疡期——并发症风险最高！**
- 饮食和活动不当极易诱发肠出血、肠穿孔

#### ④ 恢复期（第4周以后）

- 体温恢复正常
- 症状消失，食欲恢复
- 约1个月完全康复
- 约2%~5%成为慢性带菌者

### 9.5 并发症 ⭐⭐

均好发于**病程第2~3周**（溃疡形成期）

#### ① 肠出血——最常见的严重并发症

- 多发生于病程第2~3周
- 轻者：粪便隐血阳性
- 重者：柏油样便 → 鲜血便 → 失血性休克
- 诱因：进食粗纤维食物、剧烈活动

#### ② 肠穿孔——最严重的并发症⭐（主要死因）

- 好发于病程第2~3周
- 好发部位：**回肠末端**
- 临床表现：
  - 突发剧烈腹痛（先右下腹后弥漫）
  - 腹膜炎体征：板状腹、压痛、反跳痛、肌紧张
  - 肝浊音界缩小或消失
  - X线：**膈下游离气体**⭐
  - 可迅速发展为感染性休克

#### 其他并发症
- 中毒性心肌炎：脉搏增快、心律不齐、ST-T改变
- 中毒性肝炎：转氨酶升高
- 溶血性尿毒综合征（少见）
- 胆囊炎（与慢性带菌有关）

### 9.6 实验室检查 ⭐⭐

#### 血常规——注意"反常"

| 指标 | 表现 | 考试易错点 |
|------|------|-----------|
| WBC | **减少或正常**（多为3~5×10⁹/L） | 细菌感染但WBC不高——伤寒的特征！区别于一般细菌感染WBC升高 |
| **嗜酸性粒细胞** | **减少或消失**⭐ | 嗜酸性粒细胞恢复 = 病情好转的指标；持续消失 = 病情严重 |
| 中性粒细胞 | 正常或轻度左移 | — |

#### 细菌培养

| 培养类型 | 阳性率最高的时期 | 特点 |
|----------|-----------------|------|
| **血培养** | **第1~2周**（阳性率 ≥ 80%） | **确诊首选**⭐ |
| **骨髓培养** | **全病程均可阳性** | **阳性率最高的方法**⭐，已用抗生素后仍有较高阳性率 |
| 粪便培养 | 第3~4周 | 慢性带菌者随访 |
| 尿培养 | 第3~4周 | 辅助 |

**速记**：早期查血培养，全程查骨髓培养，后期查粪便培养

#### 肥达反应（Widal test）

| 项目 | 内容 |
|------|------|
| 原理 | 用已知抗原检测患者血清中的O抗体和H抗体（凝集反应） |
| 诊断阈值 | **O ≥ 1:80**，**H ≥ 1:160** 有诊断参考价值 |
| O抗体升高 | 提示**近期感染**（IgM类，出现早消失快） |
| H抗体单独升高 | 可能是**既往感染或预防接种** |
| 动态观察 | 恢复期抗体效价较急性期 **≥ 4倍升高**更有诊断意义 |
| 局限性 | 假阳性（其他沙门菌交叉反应）和假阴性（早期、用抗生素后）均可发生，**不能替代血/骨髓培养** |

**肥达反应判读技巧**：
- O和H都升高 → 伤寒可能性大
- 仅O升高 → 感染早期或其他沙门菌交叉
- 仅H升高 → 既往感染/预防接种/非特异性回忆反应
- 都不升高 → 不能排除（可能太早或免疫力低）

### 9.7 治疗

#### 一般治疗
- **消化道隔离**至体温正常后15天或粪培养2次阴性
- **卧床休息**至热退后1周
- 饮食：流质→半流质→软食，**少渣**（防肠出血和穿孔）
- 热退2周后方可恢复正常活动和饮食

#### 抗菌治疗

| 药物 | 地位 | 适用人群 |
|------|------|----------|
| **喹诺酮类**（左氧氟沙星/环丙沙星） | **首选**⭐ | 成人 |
| **头孢曲松**（三代头孢） | 一线替代 | **儿童、孕妇**、喹诺酮耐药株 |
| 阿奇霉素 | 备选 | 多重耐药伤寒 |
| 氯霉素 | 传统经典用药，现已少用 | 骨髓抑制风险（再障），需监测血象 |

- **疗程**：热退后继续用药**10~14天**（防复发）
- 糖皮质激素：仅用于**中毒症状极重**者（严重毒血症、高热不退、意识障碍），短程3~5天，地塞米松

#### 并发症处理

| 并发症 | 处理 |
|--------|------|
| 肠出血 | 禁食、静脉补液、输血；止血药；密切观察；内科保守为主 |
| **肠穿孔** | **外科手术**⭐（穿孔修补术）+ 抗感染 + 禁食 + 胃肠减压 |

#### 慢性带菌者

- 定义：排菌 > 1年
- 胆囊是主要储菌部位
- 治疗：喹诺酮类或阿莫西林，**疗程4~6周**
- 合并胆囊结石的带菌者：**胆囊切除术**（釜底抽薪）

### 9.8 预防

- 管理传染源：早发现早隔离早治疗；检出并治疗慢性带菌者
- 切断传播途径：**饮水安全**最重要（水源污染→暴发）；食品卫生
- 保护易感人群：伤寒Vi多糖疫苗（高危地区和人群）

### 9.9 鉴别诊断速查

| 鉴别疾病 | 与伤寒的区别 |
|----------|-------------|
| 败血症 | WBC**升高**、中性粒升高；无相对缓脉；血培养为其他细菌 |
| 疟疾 | 间歇性寒战高热、大汗→间歇期无症状；血涂片找疟原虫 |
| 布鲁菌病 | 也有波状热、缓脉，但关节痛突出，有牧区接触史；血培养布鲁菌 |
| 恶性组织细胞病 | 全血细胞减少，骨髓找到恶组细胞 |
| 结核病 | 午后低热、盗汗，但不呈稽留热；PPD/T-SPOT；影像学改变 |

## 附录：传染病总论高频考点

### 基本概念
- **传染源**：能排出病原体的人和动物（患者、隐性感染者、病原携带者、受感染动物）
- **传播途径**：呼吸道、消化道、接触、虫媒、血液/体液、母婴
- **潜伏期**：从感染到出现症状的时间，有流行病学意义（确定检疫期）
- **隐性感染（亚临床感染）**：感染后无明显症状但产生免疫力，是**最常见的感染类型**

### 消毒隔离
- 呼吸道传播：飞沫/空气隔离
- 消化道传播：接触隔离+粪便处理
- 血液传播：标准预防

### 常考传染病法律法规
- 甲类传染病（强制管理）：鼠疫、霍乱——仅2种
- 乙类甲管：新冠（已调整）、传染性非典型肺炎、炭疽（肺炭疽）
- 报告时限：甲类/乙类甲管 → 2小时；乙类/丙类 → 24小时
